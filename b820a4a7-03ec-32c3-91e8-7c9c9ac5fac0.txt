这时范恬上前，伸手一把抱住了夜老，她对外界发生的一切浑然不知，只诉说衷肠，“夜老，这么多年我一直在追随你的脚步，我想靠近你，又不敢靠近你，现在我不想压抑自己了，我喜欢你，你的夫人早已经去世了，不如让我照顾你吧，我想嫁给你！”
<br/>
<br/>        天哪。

<br/>        众人被雷到了，他们看着扑上去死死抱住夜老的范恬，疯了疯了，她真的是疯了！

<br/>        但是范恬感觉到了阻力，反而更加拼命的抱住了夜老。

<br/>        两方展开了拉锯战，整个A电台乱成了一锅粥。
<br/>
<br/>        这些工作人员们都像是没有见过大世面一样的看着范恬，忘记了关电台，现在实时收听率的监控已经从水平面飙到山峰去了。

<br/>        夜老的脸色很难看，他伸手将范恬给用力的推开了。

<br/>
<br/>        眼见着范恬就要将选拔赛那天的交易说出来了，夜老拿起了一杯茶，直接将茶扑到了范恬的脸上，“你好好清醒一下吧！”
<br/>
<br/>        冷水扑了范恬一脸，范恬整个一颤，恢复清醒了。
<br/>
<br/>        她刚才都做了些什么？
<br/>
<br/>        范恬抬头看了看面色铁青的夜老，又看了看厉嫣然，然后又看向那些围观的工作人员，她双腿一软，直接瘫坐在了地上。

<br/>        范恬大口大口的喘气，她浑身颤抖，像是被人推进了冰冷刺骨的寒渊里，她真的不知道自己刚才做了些什么。
<br/>
<br/>        这时“啪”一声，夜滢冲上前，终于将电台给关掉了。
<br/>
<br/>        刚才乱成一锅粥的A电台瞬间变得安静，安静到连一根针掉在地上都可以清晰的听见。
<br/>
<br/>        气氛僵滞而诡异。
<br/>
<br/>        这时“吱呀”一声，A电台的大门被推开了，有人走了进来，是邱夫人带着夏夕绾来了。

<br/>        邱夫人走上前，握了握夜老的手，“老夜，没想到这么多年过去了，你的魅力还不减当年啊，都能将一个小姑娘给迷得团团转了，厉害厉害，我真是不佩服你都不行。”
<br/>
<br/>        说着邱夫人走到了厉嫣然的面前，她拍了拍厉嫣然的肩膀，“你就是老夜的孙女吧，哎呀这真是青出于蓝胜于蓝啊，你首次电台试音何止是破了记录，简直是炸了，刚才应该是…全民收听吧。”
<br/>
<br/>        说着邱夫人又来到了夜滢的面前，“夜台长，你有这样的父亲和女儿，此刻心里一定很骄傲吧。”
<br/>
<br/>        邱夫人本来就跟夜家是竞争关系，她这轮番演讲无情的嘲笑，简直让夜老夜滢厉嫣然喉头翻涌，一口血差点喷出来。
