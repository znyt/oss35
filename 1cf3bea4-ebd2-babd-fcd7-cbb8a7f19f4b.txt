　　对这一切，叶少阳是不知道的，他一路下山之后，先去县城吃了一顿，然后就按照陈三给的地图赶路，这时候他有点后悔把马匹放了，自己又不用赶尸，没必要光脚走路，想要回去县城里买马，已经走出太远，觉得耽误时间，干脆就步行了。
<br/>　　之后几天的路程，叶少阳也没碰到什么特别的事情，而且不用赶尸体，不用照顾别人，一个人很轻松，几天之后，赶到了地图上的终点：平南县。
<br/>　　然后找到二道坝客栈，找掌柜的一打听，立刻就问到了陈三和翠云的下落：两人在这里开了两间房。
<br/>　　叶少阳在小二的带领下，去敲了一间房的门，开门的是翠云，一眼看到叶少阳，愣了一下，然后直接扑在他怀里，几乎要哭了。
<br/>　　叶少阳心中也很是感动，让她上上下下检查了一遍，确定身上没任何问题，翠云这才放心，两人坐下之后聊了一会，翠云表示他们其实也是刚到，只比叶少阳早了一个晚上，今天一大早，陈三就去交尸体去了，翠云一个人在客栈里呆着。本来失魂落魄的，没想到这么快就把他给盼来了。
<br/>　　聊了一会之后，叶少阳陡然想起一件事，从兜里取出那失而复得的几根金条，给翠云看。翠云又是一番惊喜，问他怎么得来的，叶少阳简单说了一下经过，翠云也不太懂男人之间的事情，也没多问，完全沉醉在金条的失而复得的喜悦之中，然后帮叶少阳把金条收好，这次再也不能弄丢了。
<br/>　　两人聊了一会，一起出去吃饭，在路上逛了逛，叶少阳也给翠云买了一身好衣服，又送了个玉镯子给她，翠云嫌贵，本来不要，但耐不住叶少阳一再要求，只好收下了。一路上不断埋怨叶少阳花钱多，但自己还是忍不住时不时的看上一眼……
<br/>　　回到客栈里，陈三已经回来了，并且已经从掌柜的那里得知了叶少阳到来的事，见面之后，先说了一下情况，然后提到那具女尸的苦主……
<br/>　　“这个苦主，是个邪修的法师，不过已经被废了，关于这件事，我还要像你介绍一个人，让他来说清楚，走吧，我正好约了他下馆子，见面聊。”
<br/>　　叶少阳被他神秘的样子弄得有些好奇，带着翠云一起跟他走了。
<br/>　　“今天我带你去吃个新鲜玩意，据说是四川那边的玩意，整个江西省都没有几家的，是一对四川来的夫妻逃难来到这边的开的，当地人不爱吃，不过我昨晚去尝了一下，那味道可真不错。只要你能吃辣就行。”
<br/>　　“什么鬼东西，川菜？”叶少阳首先想到这个，但是陈三表示并不是，也神秘地不愿意告诉他。
<br/>　　在街上来到一家小店面，陈三表示就是这里，叶少阳站在招牌下面，盯着上面的几个字，呆呆得站在那里，有一种傻眼的感觉。
<br/>　　“就是这了，四川那边的新式吃法，我昨晚也是第一次吃到，味道真的不错，今天也是巧，叶天师你就来了，还有翠云姑娘，今天带你们见识见识！”
<br/>　　叶少阳望着那四个大字，嘴角抽搐起来，这四个字是：陈记火锅……
<br/>　　所谓的新式吃法，原来就是四川火锅……
<br/>　　敢情这个年代，火锅还是新奇的吃法。不过一想也是，别说是民国，就算是自己小的时候，吃的火锅也跟后来的四川火锅完全不一样。
<br/>　　按照陈三说的，这种火锅的吃法在当地还没有流传开来，吃的人不多，因此客人也不多，三人要了一个包房，陈三点菜之后，出去接他说的那位朋友。
<br/>　　等了一会，火锅上餐，这个年代电还没有普及，更没有煤气之类的，用的就是碳，菜品的样子跟一百年后也不一样，比较简单，也没那么多花样。翠云听说是往里烫着吃的，感觉很是新奇。
<br/>　　叶少阳的话，在自己那个时代可是没少吃火锅，本来对火锅也没什么太大兴趣，不过这段时间也是一直没有吃，之前在店门外还有店抵触，真的看到火锅上来，还是有些食欲的，突然想到，这个时代火锅没有普及开，烧烤、小龙虾、串串这些21世纪初才流行的东西，肯定也是没有的。自己如果真想在这个时代定居的话，其实别的都不用干，光是开店卖吃的，估计也可以养家糊口，甚至发财。
<br/>　　老板在跟他们讲解这火锅是怎么吃的，翠云听得十分感兴趣。叶少阳灵机一动，等老板走后，问翠云：“姐，你有没有什么理想？”
<br/>　　“理想？”翠云新奇的目光从那些菜品上收回来，有些不解地望着他。
<br/>　　叶少阳这才想到“理想”这个词对她来说有点陌生，挠了挠头，说道：“就是说，你长大的时候，有没有想过自己将来要干什么，有没有什么想做的工作……嗯，差事？”
<br/>　　翠云听他这么说，笑道：“我一个女人家，能做什么差事，那时候就想着嫁个好人家，过安稳日子，不成想，居然做了寡妇……”
<br/>　　叶少阳赶紧打断话头，说道：“姐，不说这个，你有没有想过，不种地，开个店什么的，你菜做得这么好，有没想过开这样的饭馆？”
<br/>　　翠云想了一下，叹了口气道：“哦，说起这个，还真是。我们那镇上有饭馆的，当初我家你大哥种地，我给一个饭馆帮过工，做了一阵子厨娘，那个店老板是个浙江人，人很好，也是夫妻三个开店，说我会做菜，教过我不少浙江菜呢。”
<br/>　　叶少阳没听到重点，只抓住那四个字，重复了一遍问道：“夫妻三个？”
<br/>　　“对啊，夫妻俩，还有一个小妾，可不是夫妻三个。”
<br/>　　“好吧。”叶少阳汗颜。心想民国男人，还真的挺幸福的，一个开饭馆的都能娶两个老婆，他几乎是瞬间就到了好色的老郭，要在这个时代，他抠抠搜搜攒下那么多钱也有用处了，弄个三宫六院的，不过仔细一想，以郭大嫂的脾气，能让他娶小妾才怪。
