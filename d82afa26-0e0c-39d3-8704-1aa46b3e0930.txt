　　第557章
<br/>　　马猛惯用的兵器是一柄斩马刀，重七十二斤，他也没别的特点，就是力气特别大。
<br/>　　此刻人到绝境，竟以绝对不可能的姿势，接住了这柄重剑。双臂用力，要把重剑夺走，只要有兵器在手，他便不是任人宰割的主儿。
<br/>　　而燕不归的力气也是特别大，如果双方较力，究竟谁胜谁负，确实还说不太好。
<br/>　　但燕不归今天当真没有和马猛较力的心情，他没有这个时间。他手握剑柄，剑身往前推，马猛冷不防他来这一手，马步已经偏了。也是他反应快，使出铁板桥的功夫，硬生生避开这一剑，在地上打了个滚，虽然特别狼狈，但至少保住了性命。
<br/>　　宋寒之和张六已看出燕不归想逃，立刻围了上来。尤其是张六，燕不归上次已在他剑下走过一次，这次绝不可以让他再逃走，不然张六的脸皮往哪儿搁。
<br/>　　一剑一棍向燕不归背心袭来，来得都很凶，但凡有任何一招击中燕不归，他的性命就要搁在这里。
<br/>　　人力有时而穷，燕不归能在三大高手的围攻中玩这么久，本事也算了得。但到此刻，他当真已玩不下去了。对于身后到两招，没有防范之力，一身冷汗打湿了衣服。
<br/>　　难道自己的性命当真要交待在这里嘛，大丈夫死不死的倒不怕。可自己死在此地，带不走华正则，谁去救黑贞。
<br/>　　华正则这个时侯醒了，就看到一根大棒，一柄铁剑向自己袭来。他刚才一直处于昏迷中，此时一点也不知道是怎么回事，下意识就叫出了声。
<br/>　　“干嘛呀，杀人啦。”
<br/>　　这声音唤醒了宋寒之二人，刚才打急了，倒是忽略了燕不归身上还背着一个。此刻二人才看清楚他背的是谁。
<br/>　　“华神医，怎么是你！”宋寒之收回了剑。
<br/>　　张六可不管你是谁，打起了性子，干脆两个一块儿杀了。
<br/>　　“六爷，不可。”宋寒之用剑拦住张六。
<br/>　　这给了燕不归机会，此刻双方距离太近，出剑已经来不及。他身子以蛮牛之势撞向宋寒之，又扑向张六。
<br/>　　这是草原摔跤术的打法，无论宋寒之还是张六，都没有这方面的战斗经验。张六在海上横行无忌，水性是不用怀疑的，当初被绑住手脚，还能在水中浮游三十里，能耐可见一般。但这下盘功夫么，也就比不上擅长摔跤术的燕不归。
<br/>　　这一撞一扑，另燕不归撞出缺口。这时候，书院弟子已快到了，他哪里还敢留，纵身向山下跃去，速度极快，几个挪移，已经消失在夜幕之中。
<br/>　　程大雷眼睛一亮，这燕不归当真有一手啊。他也不耽搁，身子轻轻跃起，看好燕不归离去的方向，紧紧缀在对方身后。
<br/>　　……
<br/>　　“竟然是他？”三省阁上，夫子吃惊道。
<br/>　　“怎么不可能是他。”孟玄青道：“我一路从凉州过来，谁料半路上碰见了他。几次想要出手，但这人功夫不错，本事也是极高强，我没敢动手，怕的就是会打草惊蛇。这次你我碰上，若想除了他为公主报仇，眼下正是个机会。”
<br/>　　夫子坐在那里思索着，话不点不透，但点透之后，其实也就没什么秘密。想程大雷平日的说话办事，还有他那头黑牛……
<br/>　　夫子拍了拍手：“终日打雁，却被雁啄了眼。这泼贼当真好大的胆子，竟敢来我书院，好好好，我当真要好好会会他。”
<br/>　　“万事小心为上，贼人的本事可不容小觑。想要做这件事，还是要从长计议。”
<br/>　　夫子点点头，心中思索着如何对付程大雷。孟玄青这个时侯起身推开窗户，也未与夫子告辞，纵身离开，瞬间便消失得无影无踪。
<br/>　　夫子也不以为意，活到这把年纪，都已是半人半鬼，行事有些古怪也理所应当。现在还是对付程大雷要紧，他心中也并未小觑程大雷，如何动手，暂时还未想好。要做的，还是先想办法稳住程大雷。
<br/>　　夫子左思右想，也不睡觉了，拄着拐杖出了三省阁。
<br/>　　这个时侯，刘发财还没睡觉，在屋里转来转去。
<br/>　　“程大雷这混蛋，也忒大胆了。竟敢二探藏经楼，万一被人发现了，自己可也脱不了干系。”
<br/>　　现在他暗暗祈祷，只希望程大雷能平安无事。可他也不知道，程大雷的本事能否躲过书院无处不在的警卫。
<br/>　　咚咚咚
<br/>　　房门轻响，刘发财一机灵，就听得夫子的声音在外面响起。
<br/>　　“二位公子睡了么？”
<br/>　　刘发财眼珠一转：“睡了。”
<br/>　　吱吱呀呀，房门被推开，夫子一张老脸上堆满微笑，眼睛在房中转了一圈。
<br/>　　“只有刘公子一人，欧公子呢，是出去了么？”
<br/>　　“哎，别提了，晚饭吃坏了肚子。谁知道现在干嘛去了。”刘发财靠坑蒙拐骗为生，说谎张口便来。同时，他心中也在寻思着：夫子来干嘛。
<br/>　　夫子是什么身份，帝国天子来了，他才肯降阶相迎的主儿。如今这大半夜的，他自己过来干嘛，难道自己看着面目惊人，有人皇帝主的份儿。
<br/>　　估计这种可能性微乎其微。
<br/>　　事有反常必为妖。
<br/>　　刘发财就是琢磨不透，夫子藏着什么妖。
<br/>　　“这么晚了夫子还不休息，是……有事？”
<br/>　　“倒也没什么事啦。”夫子顿了顿：“就是偶然想起一件事，刘公子，你和那欧公子……熟嘛？”
<br/>　　熟……还是不熟了？
<br/>　　刘发财眼珠滴溜溜转着，忽然长叹一口气。
<br/>　　“公子何故长叹？”
<br/>　　“嗨，夫子不问我，我也想找个人说说。”刘发财道：“其实我和这欧公子，是路上偶然相遇，对于他的来路，我也莫不太清楚。我总觉得……”
<br/>　　“觉得什么？”
<br/>　　“这个欧公子……”刘发财又叹一声：“不像好人呐。”
<br/>　　“喔……”夫子意味深长的盯着刘发财，其实也琢磨不明白他的话是真是假。
<br/>　　何止『欧公子』看着不像好人，眼前这位刘公子看上去难道就像好人么。
<br/>　　“请问刘公子，以你所言，那欧公子还会回来么？”夫子认真问。
<br/>　　“这个……我当真说不好啊。”
<br/>(本章完)
