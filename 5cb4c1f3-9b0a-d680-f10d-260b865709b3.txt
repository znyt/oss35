　　第544章
<br/>　　作诗是个技术活，程大雷一窍不通。抄诗是个力气活，程大雷力气一直很大。
<br/>　　他当然也没有小看书院，知道这是个有文化的地方，那就不能太大意了。太一般的，肯定也是镇不住这帮书虫。
<br/>　　无奈，只有将劳驾一下本来就很忙的杜先生。
<br/>　　楚云生来了精神，像是被公主吻过的勇士，瞬间满血复活。半山有几个书生在写诗作画。楚云生跑过去借来笔墨纸砚，读书人之间还是很好说话的，对方很痛快的借给他，只是不知他要做什么。
<br/>　　楚云生来到朱漆大门前，深吸一口气，饱蘸浓墨，在大门上一字铺开。
<br/>　　万卷山的读书人是多的，想求学书院而不得的人也是多的。往日也有一些人，将自己写好的是文章词赋隔着大门丢进去，希望能遇到慧眼识珠的伯乐。
<br/>　　当然，结果一直很令人失望。
<br/>　　不过，这敢在书院大门上作诗的，确实是有史以来第一位。
<br/>　　不由得许多人都停下脚步，楚云生身后围了不少人。有两个校工想将楚云生赶开，程大雷冲刘发财使了个眼色，刘发财虽然也不想和书院作对，但面对程大雷这魔头，他没有反抗的勇气。
<br/>　　出来第一句后，周围便安静下来。
<br/>　　『纨绔不饿死，儒冠多误身』
<br/>　　楚云生是越写越亢奋，越写越激动。多年来，受人白眼遭人冷落的怨气都想了起来。琅琊城是个重视学问的地方，只要你有学问，总能混得不错。但这也表示，如果你没什么学问，琅琊城里也不好混。
<br/>　　楚云生借手中笔言心中事，笔走龙蛇，几乎是一气呵成。落下最后两句。
<br/>　　『读书破万卷，下笔如有神』
<br/>　　写罢感觉意犹未尽，一腔情绪发泄不出。因为程大雷告诉他的，就这些了，他试着在心里续了两句，但觉得都配不上前诗。他目光看向程大雷，程大雷假装没瞧着。
<br/>　　看我做什么，我也不会背啊，再说那么多典故，要是被人刨根问底问起来，让我怎么回答。
<br/>　　况且，程大雷觉得这就够了，关键是最后一句。
<br/>　　周围也是鸦雀无声，诗好不好先搁在一边，最后一句实在诛心，什么叫做读书破万卷，破的是万卷书，还是万卷书院。
<br/>　　楚云生也大概清醒过来，心里激灵一下，猛地觉察到，自己似乎是闯祸了。
<br/>　　吱吱呀呀，大门缓缓大开，一宽袍儒生立在大门前，表情有些无奈，但仍一五一十道：“传夫子话，有请楚公子。”
<br/>　　楚云生一激灵，看着洞开的大门，一时竟有些不敢相信。求了三年的大门，竟就如此开了。
<br/>　　程大雷和刘发财一左一右架起他，抬起头就往山上走，冲着书院想要拦阻的人随便支应道：
<br/>　　“一起的，一起的。”
<br/>　　楚云生仍有种身在梦中的感觉，深一脚浅一脚的往山上走。
<br/>　　“请问公子尊姓大名，在下有生之年，绝不会忘了公子大恩大德。”
<br/>　　“鄙姓欧。”程大雷一本正经。
<br/>　　“欧公子在上，请受在下一拜。”
<br/>　　现在楚云生终于对程大雷尊敬起来，都用了公子的称呼，不是刚才将二人当做下里巴人的模样。
<br/>　　“三位请留步。”
<br/>　　三人沿着山道上山，前面路上突然拦了一伙人。人人锦衣华服，手中捧着琴筝箫笛等乐器，一副要想从此过，留下买路财的架势。
<br/>　　“这是要打劫么？”程大雷摁住了剑柄。
<br/>　　为首一人先行了一礼，道：“在下钟牧山，新得一首曲子，请三位指教。”
<br/>　　程大雷一头雾水，不知什么情况。只见钟牧山一伙已经敲敲打打起来。每个人神情都很专注，摇头晃脑。
<br/>　　“这是什么路数，你懂不懂？”程大雷。
<br/>　　刘发财也皱起眉头，忽然啊呀一声，低声冲程大雷道：“我想起这人是谁了，钟牧山乃是七贤十二秀之一。最是擅长东洞箫之艺，这是来和你较量来了。”
<br/>　　七贤十二秀，是书院夫子座下最出名的几位弟子。人人都有出奇的本领，比如钟牧山就擅长乐理，俨然书院第一。而某种程度上，书院第一就能于天下第一。
<br/>　　刚才楚云生写诗要读书破万卷，又被孟夫子请上山，读书人么多是心眼小，肯定是过来找茬的。
<br/>　　一曲奏罢，钟牧山抬起头道：“在下这首沧浪破，可还过得去，请三位指点。”
<br/>　　措辞虽然客气，语气却是冷冰冰的，言外之意就是我这本事你们比得上么，比不上赶紧滚下山去。
<br/>　　“呃，蛮好，蛮好。”程大雷负手在身后：“我们就是随便上山看看，你们还奏乐相迎，还真是，还真是……”
<br/>　　“客气啦。”刘发财赶紧接口。
<br/>　　“嗯呐，太见外啦。”
<br/>　　“呃……”
<br/>　　钟牧山内心万马奔腾，那个混蛋是来迎接你们的。
<br/>　　“站住！”钟牧山突然喝出声：“三位难道就想这样蒙混过关。”
<br/>　　程大雷看看刘发财，刘发财看看程大雷，最后目光都看向楚云生。
<br/>　　兄弟，你总会些琴棋书画，该露一手的时候就露一手吧。
<br/>　　楚云生面露为难之色，尴尬的冲二人摇摇头。
<br/>　　啧，难道真是一点本事没有，想着一路躺赢。
<br/>　　看钟牧山气势汹汹的样子，不露点真本事就不能走了。程大雷啧啧一声，忽然摊开手：“与我取笛子来。”
<br/>　　如此就有了些杀人如麻的豪横劲头，钟牧山下意识将手中的竹笛搁在他手中。
<br/>　　“给我拿个新的，谁要你用过的。”
<br/>　　“呃……没有。”
<br/>　　“你瞧瞧你瞧瞧。”程大雷暗自高兴：“不是我不想表演，而是你没有准备好啊。”
<br/>　　“阁下三人既然敢号称读书破万卷，必然有出奇的本事，就该让我等见识一下，不然，书山虽有路，上山却不易。”
<br/>　　“啧。”
<br/>　　程大雷叹了一声，忽然深吸一口气，胸腔缓缓鼓起，他双手负在身后，仰天长啸。
<br/>　　初时极细，后声音渐渐高亢，如一柄孤绝的剑，穿透清风，穿破云霄，穿向苍穹最深处。
<br/>　　钟牧山耳边嗡嗡作响，有些体弱气短者已经面色发白，山上山下都是这样的声音，却没人知道怎么会事。
<br/>　　山中的野兽百鸟惊飞，周围乱糟糟一片。
<br/>(本章完)
