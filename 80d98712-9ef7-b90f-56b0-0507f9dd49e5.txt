　　轰隆！
<br/>　　大烈焚天剑，蕴含无尽血色烈焰，从天而降，这一剑，也随着剑祖荒一样，变的疯狂无比起来。
<br/>　　剑锋所蕴含的火焰，简直是要将天地的一切，都给燃尽。

<br/>　　他的剑锋便是目标一转，直接向着剑祖荒斩杀而去。
<br/>　　一抹剑光闪过，剑祖荒的身躯，顿时都被这剑光剿灭，全部破碎，一身精血，还是他的所有灵器，如今全部落入到了楚风眠的手中。

<br/>　　“不灭王拳！”
<br/>　　“圣王不败！”

<br/>　　几乎是同时瞬间，那些圣王宗弟子，同时催动无数杀招，铺天盖地之间，便是向着楚风眠轰杀过来。

<br/>　　“轰轰轰！”
<br/>　　一阵阵爆炸之声响彻而起。

<br/>　　楚风眠的身躯，从这爆炸中心，一跃而出，从他的身上，甚至都是看不到任何一点的伤痕。
<br/>　　毫发无伤。
<br/>　　“怎么可能？”
<br/>　　那些圣王宗弟子一个个都是目瞪口呆，他们如此之多人的联手之下，轰杀楚风眠，居然是丝毫无法撼动楚风眠。
<br/>　　在这种攻击之下，就算是让一尊超越生死，碎命之境的人物前来，都不敢说可以全身而退，毫发无伤。
<br/>　　可眼下，楚风眠却是做到了。
<br/>　　“你们现在一切挣扎，都是徒劳无用，全部都死吧！七大宗门的人，不配在这世界上活着！”
<br/>　　“你们都可以死了！”
<br/>　　楚风眠眼神中杀机充斥，大手一挥，便是化为了战龙之爪，横扫过去。
<br/>　　“砰！砰！砰……”
<br/>　　那些圣王宗弟子，在这战龙之爪的横扫面前，便是瞬间一个个全部被轰杀，直接打成血雾，一身精血都被楚风眠尽数吞噬。
<br/>　　“好！好，吞噬了在场的这些武者的精血，我的境界，极有可能还能再进一步！”
<br/>　　楚风眠吞噬这无数精血，脸色大喜。
<br/>　　他的战龙决，根本没有瓶颈，只需要有着足够的灵力，便是就足以突破境界。
<br/>　　这些圣王宗弟子，每一名武者的境界都是生死境巅峰的天才，在九域之中，难得一见的人物。
<br/>　　现在统统集中在这天羽神域之中，正是给这楚风眠千载难逢的机会，这绝对是楚风眠的大机缘。

<br/>　　一声暴怒之间，只见一道金色身影，猛然向着楚风眠冲了过来，一拳之间，便是冲着楚风眠轰杀过来。

<br/>　　像是这样的绝代天才，若是无法一击扼杀，越大的危机，反倒是会成为他们的动力，激发出他们的底蕴来，变的难以对付的多。
