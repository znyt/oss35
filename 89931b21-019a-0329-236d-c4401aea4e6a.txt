科学院选拔本来就是大日子，A大和T大在里面打群架这件事已经火速传遍了整个帝都城，当然夏夕绾失踪的消息也在第一时间如尘嚣一样弥漫开了。
<br/>
<br/>        大家都在猜测夏夕绾的结局，一个花季少女被几个大汉给带走了，她的清白肯定没了。
<br/>
<br/>        那些对夏夕绾有敌意的人更是将这个故事绘声绘色的传开了，一时间所有的目光都聚焦在了夏夕绾的身上，很多不怀好意。
<br/>
<br/>        陆寒霆在开车，劳斯莱斯幻影豪车疾驰在了路上，大手按着方向盘，他来电话了，是老夫人打来的。
<br/>
<br/>        陆寒霆按键接通，“喂，奶奶。”
<br/>
<br/>        “阿霆，不好了，绾绾出事了！”老夫人焦急的声音当即传来。
<br/>
<br/>        陆寒霆英气的眉心一蹙，这两天他都没有去见她，但是今天她应该在科学院的选拔赛上吧。
<br/>
<br/>        “阿霆，绾绾在科学院的选拔赛上被几个黑衣人给带走了，这件事已经满城风雨了，我觉得这件事很诡异，一定是有人借着今天的日子将绾绾推到了风口浪尖上，毁了绾绾的清誉。”老夫人迅速给出了准确的判断，然后总结道，“那几个黑衣人不会放过绾绾的，他们一定会在车上将绾绾给玷污的！”
<br/>
<br/>        陆寒霆扣着方向盘的大手倏然收紧，掌面的青筋暴跳，他整个人溢上了一层森然的寒霜。
<br/>
<br/>        “阿霆，你必须尽快找到绾绾，我将所有的线索都发送到了你的手机上。”
<br/>
<br/>        短信来了，陆寒霆点开一看，老夫人坐镇陆家这么多年，手段相当凌厉，所有的线索都发送过来了。
<br/>
<br/>        陆寒霆点开了那辆黑色面包车，那几个黑衣人就是将夏夕绾带到这辆车上然后离开了。
<br/>
<br/>        “阿霆，我已经让人追踪到了这辆黑色面包车，这车现在在西环路上。”
<br/>
<br/>        西环路？
<br/>
<br/>        陆寒霆抬头，透过蹭亮的玻璃车窗向外看去，他现在就在西环路上，这时大街对面出现了一辆黑色面包车，跟他的劳斯莱斯幻影豪车擦身而过。
<br/>
<br/>        就是这辆黑色面包车，陆寒霆一眼就认出来了。
<br/>
<br/>        “阿霆…”老夫人还在那端说些什么。

<br/>        陆寒霆伸手直接摘掉了耳朵上的耳机，猛打了方向盘，劳斯莱斯幻影掉了头，他踩下油门，豪车如火箭一样穿梭了上去。
<br/>
<br/>        有路过的车子停了下来，失声道，
<br/>
<br/>        你们快看那辆劳斯莱斯，这是不要命了啊！
<br/>
<br/>        ……

<br/>        黑色面包车里。
