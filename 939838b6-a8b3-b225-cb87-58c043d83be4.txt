        这段日子，苏漫雪一直在模仿童筱颖的一言一行，一举一动，还跟着刘婶学习做傅夜沉最爱的菜，在刘婶那儿摸清楚了傅夜沉的所有喜好。
<br/>
<br/>        总而言之，刘婶的那句话但是点醒了苏漫雪，为了傅家大少奶奶之位，苏漫雪决定豁出去了。
<br/>
<br/>        她要让自己完完全全的变成童筱颖，不管是外貌还是声音，就连一颦一笑，她都要模仿得惟妙惟肖。
<br/>
<br/>        刘婶并未发觉苏漫雪的异常反应，只是觉得这大少奶奶想通了，改了那刁钻刻薄、目中无人的性子，愿意待人和善，这是最好不过的事情。
<br/>
<br/>        人毕竟无完人，所以，每个人，都有不好的一面。
<br/>
<br/>        傅夜沉抱着童筱颖，坐上了傅氏集团大厦里他的VIP专属电梯，直达总裁办。
<br/>
<br/>        他从未如此行色匆匆过，一出电梯，他便抱着怀里的女人直奔自己办公室的二楼休息区。
<br/>
<br/>        此刻，他的眼神不再清澈，浑浊中透着一股炽热的欲望。
<br/>
<br/>        傅夜沉将童筱颖放到了自己的大床上，看着她熟睡的脸，醉意未退，粉唇晶莹剔透，饱满诱人，他的腹部下意识地紧缩。
<br/>
<br/>        他只觉自己的内心，正在慢慢地被黑暗吞噬。
<br/>
<br/>        自从她出现在傅家大庄园里，让他发现自己接错了人，而她已成为他的弟妹那一刻起，他的心就一直在痛。
<br/>
<br/>        明明一种痛，痛久了会麻木，而他却痛得生不如死。
<br/>
<br/>        筱颖，你也有错！是你让我爱上你，心痛的却只有我一个！！
<br/>
<br/>        傅夜沉俯下身来，伸出手，缓缓地解开童筱颖衣襟上的每一粒纽扣。
<br/>
<br/>        衣衫下，她的肌肤胜雪，细嫩丝滑。
<br/>
<br/>        傅夜沉看童筱颖的那双眼睛越来越深邃，眸底似乎有火焰在雀跃。
<br/>
<br/>        指尖下，衣襟往两边打开，白皙的肌肤，如玉般莹润。
<br/>
<br/>        精致的锁骨下，一件白色的内衣，裹着微微起伏的酥胸。
<br/>
<br/>        他记得，她的身子还是干净的。
<br/>
<br/>        那么，他的痛，就用她的第一次来治愈。
<br/>
<br/>        傅夜沉随之脱了自己身上的上衣，光着臂膀，往童筱颖的身上俯去。
<br/>
<br/>        她依旧睡得很沉，完全不知道自己即将成一个血气方刚的男人的“美味”。
<br/>
<br/>        傅夜沉抬起手来，爱怜地抚摸着童筱颖的脸颊，描绘着她的眉毛、眼睛、鼻子，指腹最后落在了她的樱唇上摩挲。
<br/>
<br/>        下一个瞬间，他捏住了她的下巴，微微抬起，慢慢地迎上他即将落下的吻。
<br/>
<br/>        然而，四片唇瓣仅隔那么一厘米的距离时，童筱颖却忽然睁开了双眼。
<br/>
<br/>        傅夜沉吓得一怔，动作定格在半空中。
<br/>
<br/>        “昼景，你什么时候回来的？”童筱颖睡眼惺忪，意识也很模糊。
<br/>
<br/>        她像是在梦游，又像是看错了人。
<br/>
<br/>        傅夜沉不知道她是哪一种，只是深情款款地与她四目相对。
<br/>
<br/>        “昼景，你陪陪我，好不好？我一个人好孤独……”童筱颖随之抬起双手，突然抱住了傅夜沉的脖子，往自己身上拉。
<br/>
<br/>        傅夜沉未料到她会有这样的举动，一个不留神，身子便直接压在了她的身上。
<br/>
<br/>        她柔软的肌肤，与他健硕的胸膛紧密相贴。
<br/>
<br/>        额头上渐渐地渗出了细密的汗珠，打湿了她的刘海。
<br/>
<br/>        “筱颖，我是谁？”傅夜沉心如刀割地质问。
<br/>
<br/>        童筱颖双眼迷离涣散，完全找不到目光的焦点。
<br/>
<br/>        “你是昼景啊！只属于我的昼景！你还是我的老公！这个世界上，对我最好的老公！嗯——嫁给你，我好幸福……”童筱颖说着说着，嘴边噙着一抹微笑，慢慢地合上眼睛，呓语声也越来越小，“喏——老公……我好期待……你给我婚礼的那天……我想要玫瑰花婚房……想要大大的圆床上全是玫瑰花……老公……你说……你要给我最美好的……洞房花烛夜……”
<br/>
<br/>        这是……阿景允诺给筱颖的？
<br/>
<br/>        所以，阿景才至今没有碰筱颖？
<br/>
<br/>        听着童筱颖所说的这些梦话，傅夜沉不仅仅只是心痛，更多的是心里愧疚。
<br/>
<br/>        浑浊的眼神，终于恢复了往日的冷静与犀利。
<br/>
<br/>        傅夜沉猛然起身，与童筱颖保持着距离。
<br/>
<br/>        果然，他也有内心黑暗的一面，他差一点就毁了筱颖和阿景的幸福。
<br/>
<br/>        原来，阿景不碰筱颖的原因，只是为了给筱颖一个美好的新婚洞房花烛夜。
<br/>
<br/>        那么他傅夜沉与自己的亲弟弟比起来，他是有多混蛋？
<br/>
<br/>        阿景都懂得去尊重筱颖，而他却只是……
<br/>
<br/>        或许，是他从小就被爷爷给宠坏了吧！
<br/>
<br/>        从小到大，只有是他想要的，就没有他得不到的。哪怕是在与敌人周旋的时候，他总是自信满满地将敌人“玩”得团团转。
