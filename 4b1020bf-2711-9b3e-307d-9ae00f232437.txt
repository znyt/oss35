　　尾部带有曳光管的子弹，在夜空中划出一条明亮的红色轨迹，击中了毒贩的左眼。
<br/>　　子弹削去了毒贩的半个脑袋，血浆喷射而出，毒贩被子弹带着向后摔倒。
<br/>　　少了半个脑袋的毒贩的右眼还睁着，只是他已经没有了生机。
<br/>　　有了曳光弹校准弹道，慕容飞立刻拉动枪栓，退出弹壳，推弹上膛，又是瞄准了一个毒贩。
<br/>　　子弹飞出枪膛，以一道明亮的弹道轨迹划开夜空的同时，也收割了那个被盯上的毒贩的生命。
<br/>　　普提查听到枪声响起的刹那，心中暗道不妙，他跟罗昊一样，同样没想到会突然爆发战斗。普提查是不希望战斗爆发出来的，每死掉一个人对于普提查来说都是一个损失！
<br/>　　“这些蠢货！”
<br/>　　普提查咒骂了一声，拿出藏在桌子下的手枪，拿起对讲机，对着阿卡盼大声命令道：“阿卡盼，立刻让我们的人退回来！”
<br/>　　“为什么？”
<br/>　　阿卡盼对普提查的命令感到不解。
<br/>　　“没有为什么，这是命令，立刻让我们的人退回来，不要去送死！”普提查重复了一遍自己的命令。
<br/>　　阿卡盼盯着自己手里的对讲机愣了愣神，最后还是按照普提查的命令去执行了。可是这里的毒贩并不像战术中队这边，全员配备防干扰的降噪无线电耳麦，可以随时进行联系；普提查把命令传到阿卡盼这边，阿卡盼只能扯着嗓子去传达普提查的命令，可是此时双方打得正激烈，枪声和爆炸声掩盖了阿卡盼的声音，收效甚微。
<br/>　　海面上，葛铭已经驾驶着小渔船靠近了渔村，葛铭手中拿着一挺轻机枪长短点射交替着射击，掩护着沙地上的战术中队队员撤退。
<br/>　　“烟雾弹！”
<br/>　　慕容飞和李毅峰两个人甩手扔出两颗烟雾弹，浓密的白烟飘散而出，在战术中队和毒贩之间形成一道烟雾屏障，沙地上战术中队队员迅速向后撤，翻身爬上渔船，然后继续对着从烟雾中冲出来的毒贩进行射击，葛铭重新发动渔船的柴油发动机，让渔船恢复动力后，控制着渔船离开渔村。
<br/>　　哒哒哒哒哒……
<br/>　　咻咻咻咻咻……
<br/>　　沙地上，那些紧追而来的毒贩继续用AK自动步枪对着海面上的渔船射击着。
<br/>　　子弹划破夜空，发出尖锐刺耳的破空声。偶尔有几颗子弹打在渔船上，发出“铿铿铿”沉闷的金属声。
<br/>　　“还没完没了了。”
<br/>　　林晓阳放下手中的uMP-45冲锋枪，拿出引爆器，按下了上面的引爆按键。
<br/>　　轰！轰！轰！轰！
<br/>　　那些被固定在氧气瓶上的塑胶炸药被接连引爆，因为氧气瓶的关系，令得塑胶炸药的爆炸威力比起正常情况增加了一倍不止。
<br/>　　一团一团火球冲向夜空，那些原本就腐烂严重的渔船被彻底炸成了碎片，裹着火焰木板碎片掉落在沙地上，还在燃烧着，沙地上躺着十几个毒贩，已经完全失去了行动能力。
<br/>　　当普提查来到海边，看到沙地上的惨状时，忍不住握紧了拳头。他没办法要求罗昊他们在遭到攻击的情况，不采取任何反击措施，所以普提查把这笔账也记在了布克尔特头上，如果不是他让自己去绑架那个女人，特种部队也不会找到这里来，自己不会蒙受到现在这样的损失。
<br/>　　……
<br/>　　虽然没有营救到佟千雪，但是战术中队的撤离计划并没有做出改变，依旧由葛铭驾驶着小渔船把他们送到货轮上，然后搭乘货轮离开。
<br/>　　渔船船舱中，罗昊把从普提查那边得到的情况，跟其他人进行了分析，说道：“很显然，普提查这次是帮人做事，结果惹到了麻烦，他不甘心，所以直接出卖了他背后的那个人，想要借刀杀人。”
<br/>　　“果然，这个世界上没有永远的敌人、也不会有永远的朋友，只有永远的利益。”梅庆伟感叹道。
<br/>　　“现在我们的目标已经明确，去南美洲，找到这个叫布克尔特的家伙！”
<br/>　　罗昊扫了眼自己船舱中的战术中队队员。
<br/>　　“可是我们对布克尔特这个人没有任何的了解。”齐凡皱眉道。
<br/>　　“我们不了解没有关系。”罗昊说道：“凤凰会帮我们调查清楚，大家做好准备，我们可能会在货轮上待命，然后直接出发前往南美洲。”
<br/>　　葛铭把战术中队送到停泊在海面上的货轮后就离开了，罗昊则是在货轮上跟赵天宇取得了联系。
<br/>　　“你们现在就在货轮上待命，我会让凤凰启动南美洲那边的情报网络，在最短时间内汇总好情报，并且传到你们手上。”赵天宇对罗昊说道。
<br/>　　甲板上，慕容飞站在一个集装箱边，看着漆黑的海面。
<br/>　　罗昊走到慕容飞身边，问道：“在担心佟千雪吗？”
<br/>　　“咚！”
<br/>　　慕容飞一拳头敲在身边的集装箱上，懊悔的说道：“我就不应该答应她，让她来T国！”
<br/>　　“布克尔特这次对佟千雪动手，显然是早就有所预谋，否则不可能佟千雪一抵达T国就遭到绑架。”罗昊说道：“不怕贼偷就怕贼惦记，除非佟千雪永远不出国，否则布克尔特早晚会对她下手。”
<br/>　　“凤凰那边还没有消息吗？”慕容飞对罗昊问道。
<br/>　　罗昊轻轻摇头，说道：“不过应该很快就会有消息。”
<br/>　　宋萱所建立的情报网络的工作效率比罗昊想象中的还要高，凌晨五点的时候，就有情报传到了罗昊手中。
<br/>　　“这个布克尔特在南美洲还挺有名气。”
<br/>　　罗昊把宋萱发给自己的资料，传到其他战术中队队员的PDA上，说道：“布克尔特，代号‘毒螳螂’，前美国‘绿色贝雷帽’队员，退役后，在‘黑水’安保部队做过雇佣兵，离开‘黑水’公司后，布克尔特在南美洲建立起自己的势力，种植、提炼、销售cocaine，他出售的cocaine纯度是南美洲最高的，这也使得布克尔特有着源源不断的金钱收入，让他把自己的集团越做越大。布克尔特有自己的武装组织，他的武装组织不是普提查手下那些三脚猫毒贩所能比的，其中有不少是职业军人出身！”
