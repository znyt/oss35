　　三日之后，元始便不再开口，而慈航等人还沉醉于佛道经典之中，不曾醒来。元始不急，并没有打扰他们，而是等他们自己醒来。
<br/>　　一日之后，燃灯道人最先醒来，他全身金色佛光灿灿，修为更加圆润，多了一种返璞归真的气质。
<br/>　　随后，慈航、文殊、普贤三人先后醒来，他们身上皆多了一种普度众生的佛性气质，浑身不似之前那般缥缈仙气，更像是一种拥有大智慧、大慈悲的贤者。
<br/>　　“多谢师尊成全！”四人对元始恭敬拜道。
<br/>　　“嗯。”
<br/>　　元始看向四人，突然一声大喝：“此时不悟，更待何时！”
<br/>　　随即，他挥手打出四道流光进入四人身体之中，四人俱浑身一震，似有所悟，随后气息大变。往日种种，如梦幻泡影，历历在目，有过挣扎，有过痛苦，有过收获，有过喜悦，一切经历，一切感悟，最终都化为一道感动留在心间。
<br/>　　坚定信念，对着自己便是一刀斩下，前尘种种，诸般大道，皆化为道念，随着这一刀斩下。
<br/>　　轰隆！
<br/>　　一声轰向，四道身影从四人身体之中走出，由虚幻化为真实，随即，四股强大的气息从四人身上爆发，燃灯最强，慈航三人次之。
<br/>　　显然，有元始的庇佑与帮助，他们成功斩尸突破了。燃灯成了准圣后期，慈航三人斩去一尸，成了准圣初期。
<br/>　　“多谢师尊成全！”四人内心固然欣喜，但也未曾忘记对元始拜道。
<br/>　　“嗯，一切皆是尔等之命，去吧，去完成尔等使命。”元始挥了挥手，道。
<br/>　　“是，师尊！”四人躬身退下。
<br/>　　元始屹立山崖之上，望着远处天空的白云，不知在想些什么，任由微风吹过，两肩上发丝飘摇。
<br/>　　白云悠悠，晴川历历，片刻后，白云不知去处，天空一碧如洗。
<br/>　　……
<br/>　　三日后，东胜神洲突然升起四道璀璨佛光，与此同时，四道强悍的气势冲天而起，直冲云霄，那气息之中，带着浓浓的佛性，吸引了众人的目光。
<br/>　　如今佛教已经成立，众人对佛道也已经有了一些了解，所以对着气息并不陌生。
<br/>　　在众人疑惑中，不久，一道声音响彻天地间，令三界诸神震惊不已：
<br/>　　“吾阐教燃灯道人，今于嵩山之巅立下静念禅宗，我为天僧燃灯古佛！”
<br/>　　轰！
<br/>　　三界震动，嵩山之上，一座金碧辉煌地殿堂缓缓升起，升到高空停下，如空中殿堂，其内梵音袅袅，佛光四射，照耀四方。
<br/>　　随后，一道巨大身影也顶天而起，一个身披金色袈裟，脑后神轮漂浮的身影浮现虚空，他一手托起琉璃灯，无意灯光照射四方，仿佛能够洗涤众生心中的污秽，给人一种无限光明的感觉，指导着苍生不断前进。
<br/>　　此人正是燃灯古佛！
<br/>　　在众神都没缓过气来之际，又是三道宏大声音响起，惊动洪荒：
<br/>　　“我为阐教慈航道人，今日于帝踏峰上立慈航静斋，我为地尼观自在王如来！”
<br/>　　“我为阐教文殊广法天尊，今日入静念禅宗，为文殊菩萨！”
<br/>　　“我为阐教普贤真人，今日入静念禅宗，为大行普贤菩萨！”
<br/>　　在众人震惊的目光下，三道金光闪闪的浩大身形出现于天际，慈航真人一身白色僧装，头戴白色僧巾，满头乌黑发丝向后垂下，她左手托起一玉净瓶，右手捏起一枝绿柳枝，脚踏白色莲花座，脑后还有一轮金色神轮如大日一般光芒四射，普度众生，给人一种大智慧，大慈悲的感觉。
<br/>　　普贤真人侧身坐于一六牙白象身上，右手向上抬起，掌心先前，做了个佛手指天姿势，左手持有一柄辟邪宝剑，怀抱于胸，头戴一金色圈冠，给人一种大贤大德的气息，能够令人心悦诚服。
<br/>　　文殊广法天尊盘腿坐于青色狮子身上，左手托起一株红莲，右手下垂指地，脑后一轮青色光圈，身边莲花万朵，环绕着他绽放，有一种大无畏、大无私的威严气质。
<br/>　　四道巨大的金色身影近乎占据了东胜神洲的半个天空，让所有人都能够看清楚，更能感受到其上的无上威严。
<br/>　　“观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。”
<br/>　　“舍利子，色不异空，空不异色，色即是空，空即是色，受想行识亦复如是。”
<br/>　　“……”
<br/>　　“故说般若波罗蜜多咒，即说咒曰：揭谛揭谛，波罗揭谛，波罗僧揭谛，菩提萨诃。”
<br/>　　接下来，四人开始说道，说的是筑佛道之基，亦为一教之本。如大乘佛教的《金刚经》、阐教的《玉清元始经》等，慈航静斋的立教之本为《摩诃般若波罗蜜多心经》。
<br/>　　哗！
<br/>　　这一刻，洪荒彻底沸腾了，先有接引与准提二圣于西方灵山立下佛教，如今又有阐教四大弟子于东方之地也立下佛教，这是要开启东西方佛教争夺战啊！
<br/>　　你西方立下佛教，我东方一样可以立，就看谁更厉害，谁便是这佛道之主。
<br/>　　“东方竟然也立佛教了，而且听起来比西方的也不差多少，甚至在一些方面比起西方佛教更有独到之处。”
<br/>　　“什么，东方也立了佛教！如此也好，我等东方之人便不必经历万险去西方寻道了！”
<br/>　　“你傻啊，东方佛教能够与西方比吗，西方可是有二圣啊！”
<br/>　　“你刚才没有听到吗？东方佛教的创立者可是阐教门下，他们背后可是洪荒第一圣元始天尊啊，又岂会比西方差？”
<br/>　　“那也是，静念禅宗与慈航静斋刚刚成立，正是光收门徒的时候，我等当立即前往，说不定能够被收为弟子。”
<br/>　　“慈航静斋只收女子？我等女性修士也有求佛之心，慈航静斋正好适合我们。”
<br/>　　“……”
<br/>　　顿时，洪荒一片议论纷纷，众多修士，特别是东方散修，都纷纷朝着静念禅宗与慈航静斋飞去，想要拜入门下，成为弟子。
<br/>　　慈航静斋与静念禅宗相距不远，两者相互依存，本是同出于阐教，如今又同为东方佛教，关系自然非比寻常。
<br/>　　“出！”
<br/>　　燃灯古佛翻手取出三十六颗定海神珠，神珠璀璨，如同三十六颗浩大明月升空，演化三十六方诸天，镇压静念禅宗的气运。
<br/>　　此三十六定海神珠可比二十四颗厉害多了，而且品级也提升了，二十四颗只是上品先天法宝，三十六颗可是顶尖的极品先天灵宝，镇压气运比西方佛教的九品金莲强多了，毕竟它可是与十二品金莲同级的。
<br/>　　同时，慈航静斋斋主观世音亦取出一座九品净世白莲，扔到帝踏峰上空的气运池中，镇压气运。
<br/>　　两者都是东方佛教，以一件法宝镇压气运即可，两件更是强大，能够保证气运不失，汇聚更多东方气运。
<br/>　　昆仑山上，元始神秘一笑，这两件法宝自然是他给慈航等人的，他倒是想看看准提与接引二人如今该如何应对。
<br/>　　如此一来，未来究竟是谁取谁的经还不一定呢。
<br/>　　……
