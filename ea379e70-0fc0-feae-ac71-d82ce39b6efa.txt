　　“交出剑鼎，今日不杀你。”
<br/>　　听到这一句话的一刻，周围不少七大宗门一方的武者吗，都为之一愣。
<br/>　　旋即在下一刻，更是响起了一声声的爆笑声音。
<br/>　　“这小子是得了失心疯了吧。”
<br/>　　“太自大了。”
<br/>　　无数目光看向楚风眠，都如同是在看向一个疯子一样。
<br/>　　在楚风眠面前的这荒鎏金，可是一尊真正的半步天人境强者啊。
<br/>　　而楚风眠哪怕是再强，得到了在多的奇遇，也不过只是九劫古帝罢了。
<br/>　　纵然是在历史之中，都没有过任何一尊九劫古帝，曾经斩杀过半步天人境武者的。
<br/>　　“小子，看来你是在自己找死！”
<br/>　　那荒鎏金，听到楚风眠的话，脸色也是一下子阴沉了下来，一双目光看向楚风眠，眼神之中杀意暴起。
<br/>　　如今在天下之间，已经是没有了天人境武者的存在，半步天人境，已经是巅峰了。
<br/>　　自从这荒鎏金，踏入了半步天人境之后，还是第一次被楚风眠这般出言轻视。
<br/>　　不过在这愤怒之下，荒鎏金的心中也是无比震惊刚刚楚风眠所说的话。
<br/>　　曾经的他，被扔到了无尽海沟的深处，就是因为他曾经偷偷潜入到了剑道门之中，盗取了剑道门至宝之一的剑鼎。
<br/>　　后来荒鎏金虽然是被抓住，但是剑鼎已经是被悄悄藏了起来，后来等到他从无尽海沟之中离开，依靠这剑鼎，才成功踏入到半步天人境的。
<br/>　　他得到剑鼎这个秘密，天下之间，几乎已经无人知晓了，哪怕是荒古门之中，都没有人知道。
<br/>　　曾经知道的剑道门弟子，可都是在万年一战之中，全部陨落了，这剑鼎也是他最大的底牌，最终的秘密。
<br/>　　现在居然是被楚风眠一把点破了出来，让他心中也是一阵惊慌，不过在这荒鎏金心中，更多的却是杀意。
<br/>　　剑鼎，可是曾经剑道门之中的至宝之一，哪怕是对半步天人境级别的强者来说，都有着大用，这个秘密他可不愿让楚风眠暴露出去。
<br/>　　“死！”
<br/>　　轰隆！
<br/>　　这荒鎏金想到这里，便是不再犹豫，突然出手了，他的手心之中荒古之力再度凝聚，化为了荒古长戟，猛然向着楚风眠斩杀而去。
<br/>　　一出手便是绝对的杀招。
<br/>　　如今被楚风眠道破了剑鼎的秘密之后，他已经不再是打算镇压楚风眠了，而是要一举杀了楚风眠。
<br/>　　只有杀了楚风眠，才可以让这个消息完全的封闭，以绝后患。
<br/>　　“破灭天地！”
<br/>　　轰隆隆！
<br/>　　荒古大戟之上的力量，被催动到了极致，比起那五行神刀，还要更加强横的历练，从天而降，向着楚风眠轰杀而来。
<br/>　　从这荒鎏金的背后，更是显现出了一副场景，那是一个无比古老的时代，曾经的荒古时代，群兽争霸，参天大树，足有万丈之高。
<br/>　　山峦叠起，最为古老的时代。
<br/>　　就是在这一副图像之中，源源不断的荒古之力爆发出来，纷纷加持在了那荒古大戟之上，以雷霆之势，灭杀楚风眠。
<br/>　　“不愿意交出剑鼎，那你就死吧！”
<br/>　　楚风眠眼神冰冷无比，这剑鼎，是剑道门的至宝，决不能落入到其他人的手中。
<br/>　　这荒鎏金不愿意交出，那楚风眠就杀了他，在将剑鼎夺回来。
<br/>　　“剑出！”
<br/>　　搜搜！
<br/>　　从楚风眠的背后，天涯剑匣猛然打开，一道道剑气，旋即飞了出来，围绕在楚风眠的身边，开始凝聚这，足足的化为了一道万丈剑气，冲天而上。
<br/>　　跟这荒古大戟的力量，猛然碰撞。
<br/>　　这一次的交锋，爆发而出的力量，比起上一次交锋之时爆发出来的力量，还要强横无数倍，一时间地动山摇。
<br/>　　毕竟上一次交手，还有着几分试探的味道。
<br/>　　但是这一次交手，双方打出的，可都是绝对的杀招，没有丝毫的保留，就是要将对方所灭杀。
<br/>　　不管是剑道门一方，还是七大宗门一方的武者，如今都只能够是爆发出灵力，来死死的守护这自己的身躯，免得被这余波所波及到。
<br/>　　这种余波，哪怕是九劫古帝稍微碰到一点，都会被无情的轰杀。
<br/>　　强横的力量，足以是将一身的精血都给震碎，让人就连重塑肉身的机会都没有。
<br/>　　半步天人境强者的交手，是帝尊境，根本无法插手其中的。
<br/>　　刹那之间。
<br/>　　万丈剑气，荒古大戟，已经是交锋了无数次，每一次碰撞，都爆发出极为强横的力量来。
<br/>　　一时之间，更是难以分出胜负来。
<br/>　　“好强横的力量。”
<br/>　　“这小子居然真的可以跟荒鎏金交手？”
<br/>　　“此人不过刚刚踏入九劫古帝，可是这力量，绝对是半步天人境级别的力量了。”
<br/>　　“太恐怖了，此人才不过是九劫古帝境界，就有着如此恐怖的力量，一旦是让他踏入半步天人境，那还不是无敌了？不行，此人必须要灭杀！”
<br/>　　无数七大宗门强者，看着天空之上的交锋，心中都有着各异的想法。
<br/>　　有的震惊，有着惊讶，有的则是充满杀意，各怀鬼胎。
<br/>　　不过他们所有人，也都没有出手，而是静静的观察这。
<br/>　　这个时候谁若是出手，贸然插手其中，那可能就会被当做炮灰，直接死在了二人的余波之下。
<br/>　　他们纵然是出手，也是要等待楚风眠跟这荒鎏金，拼到两败俱伤的地步的时候，在突然出手，趁机落井下石。
<br/>　　在历史记载之中，九劫古帝正面轰杀半步天人境的记载，的确是没有。
<br/>　　但是半步天人境，死在九劫古帝手下的故事，可是有着不少。
<br/>　　一旦是陷入到危机之中，趁火打劫，可是有着极大的可能，可以轰杀半步天人境强者的。
<br/>　　楚风眠的身上，可是有着大机缘，能够以九劫古帝的境界，就拥有如此可怕的力量，楚风眠身上的奇遇机缘，不言而喻。
<br/>　　谁能够杀了楚风眠，得到了楚风眠的奇遇机缘，便是一步登天。
<br/>　　只要是楚风眠露出破绽，陷入败势，他们便是会以雷霆之势出手，灭杀楚风眠，来瓜分好处。
<br/>　　不过如今，却都是在静静的围观这，都在等待着机会。
