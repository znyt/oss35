　　第721章
<br/>　　崔白玉盯着程大雷油腥腥的嘴，脏兮兮的大手，极力忍住想要呕吐的冲动。她终于明白程大雷要带自己去长安的原因……
<br/>　　他是否和李乐天达成某种交易暂且不知，但至少能够确定，沿途侮辱自己必是他的目的之一。
<br/>　　渴了要自己端水，饿了要自己温干粮，他走累了还需自己给他烧洗脚水……至少，他还没有令自己暖被，但估计也是顾忌周围人多眼杂的原因居多。
<br/>　　崔白玉收敛眼中的杀意，摇头拒绝程大雷的『好意』。
<br/>　　程大雷倒也没有生气，把肥硕的鸡腿塞到自己口中大嚼特嚼。程大雷、刘发财、徐神机其实都没什么文化，言语粗鄙，坐在那里一边喝酒吃肉一边胡吹海侃。
<br/>　　鱼洞真算是崔白玉唯一的同类，但她就像一个木头人一样，不会给出任何反应。对崔白玉来说，单单与几人同桌而食已经是莫大的摧残。
<br/>　　“几位客人，饭菜可用得顺口，如果有什么不满意的，我可以让厨房重做。”
<br/>　　陈德山拎着一坛酒进来，脸上挂着笑容。
<br/>　　“老伯客气了，逆旅之人承蒙老伯照顾已经感激不尽，怎还敢挑三拣四。”程大雷站起来拱手道。
<br/>　　崔白玉及时挂上黑纱，退到一旁去了。不过，即便只是匆匆一瞥，也令陈德山目不转睛。但诧异的眼神一闪而逝，随即便恢复了正常。
<br/>　　面对崔白玉的美貌，只失神片刻是极不容易的事情，如此也能看出陈德山的不简单。
<br/>　　他拎着酒坛坐下，冲程大雷道：“这是庄里自己酿的杏花醉，拿来与几位客人尝尝。”
<br/>　　“客气了，委实客气啦。”
<br/>　　酒封拍开，便是一阵清香溢出，程大雷都不自觉抽了抽鼻子。刘发财和徐神机口水都快流了出来。
<br/>　　程大雷在桌子底下一人踹了他们一脚，还真是不给自己长脸呐。
<br/>　　陈德山一一为几人斟满酒：“不知几位客人如何称呼，刚才见面匆忙，还没来得及请教。”
<br/>　　“姓牛。”程大雷一拱手：“牛三斤的牛，这次去京城是找几个老朋友。”
<br/>　　“看客人的样子，倒是有股军武英气。”
<br/>　　“猜对了。”程大雷一拍桌子：“是凉州边关守将，日子过不下去了，去京城讨饷。”
<br/>　　程大雷这言语不祥不实，陈德山倒也没有追问。程大雷倒是多问了对方几句。
<br/>　　问过之后方才知道，这陈德山是方圆百里有名的富户，外号陈百万。家中虽然没有良田千顷，但其实也不差多少，养着三百多号人。而且名声很好，旱时舍粥，涝时舍米，行脚过路客人，一般都会收留。
<br/>　　闲聊片刻，陈德山也没有多待，便告辞离开。。
<br/>　　他刚走，刘发财和徐神机便同时扑向那坛酒，一边往外倒一边问：
<br/>　　“这酒当真不错，对了，大当家怎么不问问他家里遇着什么事了？”
<br/>　　程大雷道：“问也是白问，你以为他会告诉我们。还没看明白，这是过来探底的，明显是信不过我们。”
<br/>　　陈德山离开院落，还没走几步陈忠便迎了过来，将一根手杖递到他手中。
<br/>　　“老爷，查探出什么没有，这伙人会不会是一起的？”陈忠低声问。
<br/>　　陈德山摇摇头：“看上去只是过路人，不像是搅合进这件事里的，只是他们的来历，还是没看出来。看着有些血腥气，不像是好人。”
<br/>　　这不单单说的是程大雷，而是算上徐神机、刘发财、鱼洞真，即便是貌美如玉的崔白玉，五个人统统不是好人。
<br/>　　程大雷用罢了饭，夜渐渐深了，除了程大雷所住的院落，整个陈家庄是灯火通明，门口挂着两只大红灯笼。
<br/>　　大概是二更十分，十几匹快马从远处奔驰而来，其中一人弯弓射箭，飞箭将门前两只灯笼射灭。
<br/>　　“姓陈的出来。”
<br/>　　一声喊无人应答，从队伍中走出一人，他背上扛着一柄大锤，走到门前忽然发力，三两下便将大门砸的稀巴烂，扛着大锤走入大门。
<br/>　　十几人驰马破门，迎接他们的是上百号陈家壮汉。
<br/>　　都是正当年的男人，手里拿着铁叉，棒槌，少数几个人拿着朴刀铁棍。
<br/>　　陈德山仍旧是一身员外服，他拱拱手道：“荆棘岭的好汉，金银已准备好，老汉双手奉上，只求饶我陈家上下一命。”
<br/>　　为首一人道：“陈老汉，不要废话，我们大当家相中你家姑娘，要你做他的老丈人。新娘子在哪里，我们老大等着接回去入洞房呢。”
<br/>　　陈德山再次抱拳：“老汉膝下只有一女，要她养老送终，求各位好汉高抬贵手，要钱要粮尽管开口。”
<br/>　　“呸，你当我们是要饭的么。”为首自然啐了一口道：“我们大当家说了，三更天没接到人，便屠光陈家庄，今天先给你送了终。”
<br/>　　陈德山抱在一起的手终于放开，向旁边一人道：“陈家愿以三千两白银，请冯先生出手。”
<br/>　　事情要从三天前说起，这伙入院的匪人乃是附近荆棘岭的好汉。荆棘岭上盘踞着一伙山贼，为首之人外号小白龙，使两口刀，杀人越货凶恶得狠。
<br/>　　也就是踩盘子的时候看见了陈家的小姑娘，便动了心思，差人送信要将陈家小姑娘聘走。
<br/>　　陈德山心急如焚，为的就是这件事。他家里养着三百多号人，但也敌不过小白龙的手下。
<br/>　　于是重金请了这位姓冯的高手，三千两银子听上去不像一个大数目，但在帝国，能换一百名模样不错的丫头，在京城最好青楼挥霍一年，那削铁如泥，价值连城的名刀宝剑，也差不多能买个七八柄。
<br/>　　程大雷、刘发财、徐神机、崔白玉、鱼洞真趴在屋顶，偷窥着这一幕。
<br/>　　“瞧瞧，原来是抢媳妇来的。”程大雷道。
<br/>　　“大当家，咱们要不要出手？”
<br/>　　“再瞧瞧，看看这剑客怎样本事，等下再出手也不迟。也不知那新媳妇生得怎么模样，值得如此大费周章。”程大雷道。
<br/>　　冯家剑客抱剑而立，望了望荆棘岭再看看陈德山，口中吐出两个字。
<br/>　　“加钱。”
<br/>(本章完)
