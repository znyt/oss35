        这个男人虽然跟那八爷一样，戴着假面眼镜，但是无论是脸型还是身形都和傅夜沉一模一样。
<br/>
<br/>        但是，他的眼神里透着陌生和冷冽，让周遭的空气瞬间凝固。
<br/>
<br/>        童筱颖的嘴唇抖了抖，“大哥”这两个字堵在喉咙里，怎么也叫不出来。
<br/>
<br/>        “阿英，这位是刚从国外办完大事回来的四爷，咱们这四爷，可是那种只要是出高价钱连女人和孩子都敢杀且毫无底线的无情杀手，今晚你可要好好地给我伺候着，不然小心四爷杀了你噢！”八爷笑着命令道。
<br/>
<br/>        童筱颖默不吭声，故作乖巧地点头。
<br/>
<br/>        刚刚她还误以为这位四爷会不会是大哥，结果八爷接下来的话，让她心痛得倒吸了口冷气。
<br/>
<br/>        “四哥，你把傅夜沉弄死了，倒是让我心里感到空虚了，我自制了好多定时炸弹，现在都没有对手帮我拆了。咱们集团的大佬们就急着要集聚在一起在今晚开重大会议。你说，这大会上，一爷、二爷和三爷，会不会给你颁发奖状？”八爷意味深长地打趣。
<br/>
<br/>        八爷的话到这，童筱颖猛然想起，三年前，自己在商场里不知道被谁绑了定时炸弹在腰间的事情。
<br/>
<br/>        那个时候，她以为她死定了，结果最后是大哥傅夜沉出面帮她拆的定时炸弹。
<br/>
<br/>        所以说，八爷一直在暗中跟大哥傅夜沉作对？
<br/>
<br/>        而现在，更重要的一点是，这个抱着她坐在大腿上的男人，竟然就是她想要报仇的对象！
<br/>
<br/>        “小八也可以给我出高价钱，去杀你恨的那个人。”四爷的声音低沉却摄人心魂。
<br/>
<br/>        “我出一千万，你帮我杀了一爷，你敢吗？”八爷笑了笑。
<br/>
<br/>        四爷淡定道：“他是我们的老大，自家兄弟，不能相互残杀，这是规矩，小八忘了吗？”
<br/>
<br/>        “你们这几个女人继续啊！一个个愣着做什么！”八爷一脸不悦地瘪了瘪唇，开腔呵斥着跪在地上的女人，索性拿她们出气。
<br/>
<br/>        童筱颖讷讷地开口，对面前这个四爷说道：“四爷，如果我能徒手把八爷的名表从那蜈蚣群里取出来，我不要钱，只求四爷替我向八爷求个情，放了这里的女人，如何？”
<br/>
<br/>        “小八，你这送给我的女人，似乎还是个同情心泛滥的女人。”四爷的薄唇扯了扯，黑眸凝视着童筱颖，放在她腰肢上的大手突然紧握，眼底透着邪肆与轻蔑。
<br/>
<br/>        八爷饶有兴趣地打量着童筱颖，但心里却在狐疑这个女人的声音，听起来不像是他经常“光顾”的阿英。
<br/>
<br/>        童筱颖瞥了一眼八爷，见他正打量着自己，又下意识地看向四爷。
<br/>
<br/>        四爷却突然起身，将童筱颖的身子顺势捞起，横抱在怀中。
<br/>
<br/>        “小八你接着玩，我带这个女人上去解决一下生理需求。”
<br/>
<br/>        八爷笑了笑：“那你去吧！”
<br/>
<br/>        童筱颖顿时惊愕地瞪大了眼睛，但是现在却不是她反抗的时候。
<br/>
<br/>        四爷抱着她坐电梯上了楼，到了一间606号房门口，他将她放了下来，一手抓住她的手腕，一手从衣兜里掏出来房卡，刷卡开门。
<br/>
<br/>        当他将她拽进屋内后，房门一关。
<br/>
<br/>        童筱颖便开口说道：“我伺候四爷洗澡吧！”
<br/>
<br/>        “看样子，你比我还心急。”四爷勾了勾唇。
<br/>
<br/>        童筱颖一脸淡定地回答：“我只是在遵从八爷的命令而已，我现在去给您放洗澡水。”
<br/>
<br/>        四爷随之放开了她。
<br/>
<br/>        童筱颖进了浴室，走到浴缸旁，蹲下身去，拧开水龙头调水温。
<br/>
<br/>        四爷忽然出现在她身后，高大的身影，将童筱颖的身子完全笼罩。
<br/>
<br/>        童筱颖不敢回头，心里却紧张害怕得打起了小鼓。
<br/>
<br/>        四爷并张开了双臂，命令道：“你帮我脱衣。”
<br/>
<br/>        “好！”童筱颖随之起身，心不在焉地应着。
<br/>
<br/>        她心里一直在想，如何逃脱的计策。
<br/>
<br/>        童筱颖转过身去，面对着站在自己面前，身形高大的四爷，他身上这身昂贵的黑色风衣，衬得他更加矜贵迷人，就像傅夜沉给她的感觉一样。
<br/>
<br/>        她慢条斯理地替他脱衣服，当她看到他腰间别着一把手枪时，冷不丁地打了个寒颤。
<br/>
<br/>        或许是因为一时脑热，又或许是因为慌不择路。
<br/>
<br/>        童筱颖不知道哪里来的勇气，趁着替他解开皮带的时候，拔走了他的手枪，直接指向了四爷的脑门。
<br/>
<br/>        她的脸色已经惨白，持枪的双手不停地颤抖着。
<br/>
<br/>        四爷却异常冷静地直视着她。
