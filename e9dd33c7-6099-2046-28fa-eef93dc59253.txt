　　“殿下，事情好像有些不大对头，这岛上应该有我们的士兵在驻守，按说在这个位置应该能够看到岛上他们为我们指明航向的灯光。”李震再次出现在李佑的身边，指着远处的暗影说道。
<br/>　　李佑摇摇头，似乎想把脑子里所有不相干的东西都甩出去，片刻之后看着李震问道：“你的意思是什么？”
<br/>　　“派人上岛看看，此前我们经过的岛上都有我们的人，按地图所标识的情况来看，这里应该也有，但不知道他们为什么把火给熄了。”李震皱着眉头说道。
<br/>　　“有把握么？现在可是晚上，岛上很可能并不安全。”李佑有些不大确定的说道：“而且我看这岛似乎不小，你又如何能够确定岛上驻军的位置在什么地方？”
<br/>　　李震再次扭头看了一眼远处的海岛，摇头说道：“没有办法确定，最多只能是上去碰运气，但岛上的驻军并不是只有一个，总不可能全都睡着了，所以我认为最大的可能是岛上的驻军最大的可能是已经遭到不测，所以才会出现现在这样的情况。”
<br/>　　“那就更不能现在上去。”李佑再次否定了李震的想法，沉声说道：“这么大的岛，上面一定会有土著，我们这么大一支舰队到来必然会惊动他们，如果这个时候我们登岛，很可能去多少人就扔里多少人。”
<br/>　　“那……”
<br/>　　不等李震说完，李佑已经摆手打断了他：“等吧，等到天亮，天亮之后派大部队下船去探个究竟，今天晚上加强戒备，全员休息。”
<br/>　　“是……”虽然心有不甘，但李震还是咬牙答应了下来。
<br/>　　眼下这个时候正是多事之秋，老头子在外海生死不明，李佑认为这个时候最好还是谨慎一些，虽然探明岛上的情况很重要，但是人员的安全方面更加重要。
<br/>　　……
<br/>　　翌日，天刚蒙蒙亮，数艘小船已经被放到海里，由一批精锐‘海狼’划着，向岛上靠了过去。
<br/>　　那岛很大，昨天晚上李佑他们因为战舰停靠位置的关系，并没有发现，而等到‘海狼’登上海岛之后才发现，这岛的大小怕是足有数十里方圆。
<br/>　　而李佑他们并没有登岛，留下数艘战舰支援登岛的‘海狼’之后，他们便拔锚起航，向着下一个有人驻守的海岛而去。
<br/>　　但让人意外的是，自从那天晚上之后，所有他们经过的岛屿，驻军全都消失的一干二净。
<br/>　　三天之后李佑不得不再次与李震凑到一起，考虑起接下来的行动方案。
<br/>　　“李兄，已经三天了，根据‘海狼’在岛上的探查情况来看，我们的人无疑是被一些有心人处理掉了，这一点你不否认吧？”
<br/>　　李震点点头，十分确定的说道：“岛上有过打斗的痕迹，虽然经过这么长时间已经看不出来什么，但是在一些树木和藤蔓里面，我们还是找到了不少的三棱箭头。”
<br/>　　李佑显然已经知道了这一切，但他需要知道的却并不仅仅是这些，在经过短暂的犹豫之后，他沉声问道：“那么我们接下来怎么办？这里显然已经有人清理过了，我们是否还要继续这么摸索前进？你给我一个准确的答复。”
<br/>　　此前的三天，北海舰队整整探查了八座大型岛屿，在那些岛上都发现了大唐驻军曾经在岛上驻守的痕迹。
<br/>　　可是就在数月之前，这些人全都消失了，岛上除了有一些零散的战斗痕迹之外再也没有其它证据可以证明驻军的存在。
<br/>　　在如此茫茫大海之上，死几个人再正常不过，可是如果全都死了就显的有些不正常了。
<br/>　　所以说接下来是否还继续查已经没有太大的必要，如果不是李佑担心跟老头子一样遭遇敌人的埋伏，他早就已经下令舰队全速前进了。
<br/>　　李震其实比李佑还要着急，不过越是到了这个时候就越需要冷静的头脑，他清楚的知道如果两个人都冲动的话，后果一定十分可怕。
<br/>　　于是在沉思良久以后，李震缓缓说道：“殿下，我们可以试着分析一下整件事情。”
<br/>　　“你说！”李佑点了点头。
<br/>　　“殿下，您认为以大食人或者南海其他小国的实力，想要困住太上皇和南海舰队整整十二舰铁甲舰需要多少兵力？”李震揉着自己的下巴，像是在问李佑，又像是在问自己。
<br/>　　李佑的眉头皱了皱，心中隐隐有了一份答案，但想了想之后却没有马上回答，而是看着李震反问道：“如果是你呢？你觉得多少兵力才能困住你？”
<br/>　　李震并没有回避李佑的眼神，两人就那么对视着，良久之后才听他说道：“如果是物资装备充足的情况下，打硬仗没有二十倍的兵力连船上的油漆他们都别想碰掉一点，但如果是在物资匮乏，敌人又不想硬碰硬只是牵制的话，十倍兵力就可以将我牵制住。”
<br/>　　李佑再次反问：“十倍的兵力？还是十倍的战船？”
<br/>　　“十倍的兵力！”李震确认的说道。
<br/>　　大食人的战船远远要小于大唐钢铁战舰，十倍的战船兵力充其量也就是与大唐的军力相等，所以按照李震的推算来看，十倍兵力那就是百倍的战船或者更多的战船。
<br/>　　老头子那里有钢铁战舰十余艘，百倍也是是千余，大食人有那么多的战船？李佑不相信！
<br/>　　但虽然如此，却有一点可以肯定，那就是大食人一定把全部的兵力都投入到了围殴老头子的战斗之中。也就是说，大食人根本就没有留多少后手来应付大唐接下来的援兵，而且就算是有，也不足为惧。
<br/>　　这样的结论如几同时出现在李佑和李震的意识之中，使得两人颇为兴奋的对视了一眼。
<br/>　　李震一拳砸在自己的左手心里，凝声说道：“干吧殿下，接下来不用查了，我们只要照着海图一路走下去就好，敌人应该没有余力来阻止我们了！”
<br/>　　“好，你去通知舰队汇合，咱们马上出发！”李佑没有任何犹豫，十分痛快的点头同意了李震的行动方案，在整个舰队汇合之后，全力向南面杀了下去。
