　　第494章
<br/>　　程大雷双手捧着杯子，低头抿了口热茶，最可怕的敌人永远是你无法了解的敌人，如果聂隐娘是什么会妖术的巫婆，程大雷一丁点不会觉得奇怪。
<br/>　　阿喜的脑袋已经不够用了，纵然程大雷平时就透着有些奇怪，但今日的确奇怪得有些过分。
<br/>　　“这个女人是不是对大当家很重要？”阿喜认真问。
<br/>　　“当然很重要。”程大雷叹口气：“所以你们一定要认真对待，千万不可麻痹大意。”
<br/>　　阿喜点点头，觉得自己明白了。的确，以大当家的本事，这世上能伤到他的人已不多。唯一能对大当家造成威胁的，也只有女人了，还定是个倾国倾城的女子。怪不得大当家这些日子透着奇怪呢，原来是患了相思病。
<br/>　　“大当家放心，无论她是谁，我一定找到她，把她送到大当家面前。”阿喜拍着胸脯保证。
<br/>　　“不，千万不要。”程大雷大惊失色，万一那聂隐娘会什么十步绝杀剑之类的，一剑将自己灭了怎么办。
<br/>　　“这……”刚觉得有些明白的阿喜，现在又有些糊涂了：“不知为何？”
<br/>　　程大雷顿了顿，似乎不经自己辨认，确实也不好确认这聂隐娘的身份。他叹了口气道：“算了，有觉得可疑的，还是送到我这里吧，不过，一定要小心。”
<br/>　　“放心，绝对出不了岔子的。”
<br/>　　阿喜心想，大当家看中的女人，怎么会让她在自己手下出事，定然是认真看待。他离了城主府，立刻将喜字队的叫到一起开会，只是说大当家中意某个神秘女子，现在大当家把这任务交给喜字队了，大家一定要将这任务办妥，帮大当家找到这女子。
<br/>　　之后，消息便传开了，只说程大雷是为了一个女人害了相思症，才大门不出二门不迈。如和珅、徐神机等人还过来看望程大雷，说什么『大丈夫要志在千里，不要为一个女人磨损心志』之类的话。
<br/>　　程大雷听得是一头雾水，后来才算是明白怎么会事，恨得是咬牙切齿，将阿喜叫过来狠狠训了一顿，山寨里的风言风语才算是平息下去。
<br/>　　但喜字队的确在城门口开始盘查，不分老弱，只要是女人都要细细查问底细，因为程大雷担心这聂隐娘会什么易容术之类的，所以查得很仔细。尤其是独身女子，必须相熟的人同行，方可入城。
<br/>　　这一日，沿着长道过来一个女子，她穿着麻衣，一路风尘仆仆，在快要接近蛤蟆城城门时停下，向远处城门方向眺望。
<br/>　　只见进出城池的人络绎不绝，城里的人想要出来还是很容易的，但城外的人想进来就困难多了。住在城外的平民，免不得会往城里买些生活必需品，或将草鞋、小菜之类的东西挑到城里贩卖，现在进城要被卫兵仔细盘问很久。
<br/>　　女子没有继续往前走，而是转了个方向离去。
<br/>　　夜来，蛤蟆城的城门已关上，这时代已无什么娱乐活动，城里城外一片安静，偶尔有几乎人家点灯的，也是十分罕见。
<br/>　　城门之上，有来来往往提着灯笼巡逻的卫兵，在程大雷『闭关』之后，城墙之上也加强了警戒。
<br/>　　巡逻的卫兵并不知道，城墙下多了一个黑影，她立在墙下，向上望了一眼，约莫是在估计高度。随之身子轻轻跃起，整个身体攀在城墙上，然后又借力一跃，便又是三五步的高度。
<br/>　　如果白元飞看到这一幕，怕会大惊失色。他也曾夜爬凉州城，但绝对做不到像这女子一样轻盈。
<br/>　　她像夜里的孤燕，双翅如双翼，划过空气不发出任何声音。
<br/>　　在两队卫兵擦肩而过的光景，女子轻盈飞过，跃入了城中，身子在空中一个转折，落在一户人家的房顶上。
<br/>　　她微微抬头，用视线辨别城主府的位置，如今这个时代，大部分城池的建筑布局都差不多，城主府都位于城池的中心，蛤蟆城也并不例外。
<br/>　　城内也有巡逻的卫兵，砍出五步一哨，十步一岗，如此可见程大雷对自己安全的在意。
<br/>　　但女子在屋顶上行走，脚踩在瓦片上时，没有一丁点动静，巡逻的卫兵在街道上警戒，却并不知头顶有人行过。
<br/>　　神不知鬼不觉，她已迁入城主府，接下来的事情令她有些为难。她能第一时间找到城主府，却未必知道程大雷住在那间屋子里。
<br/>　　这个时候，屋顶下院子里有两条恶犬突然疯狂的叫了起来。这女子心中一惊，随手抓起一片瓦捏碎，嗖嗖两指弹出，两只恶犬立刻没了声息。
<br/>　　为了保证自己的生命安全，程大雷在自己住的院子里养了两条狗。聂隐娘的确没发出任何声音，堪称神不知鬼不觉，然而她身上的气味却无法掩盖，还是被这两条恶犬发现了。
<br/>　　虽然她动作很快，第一时间便解决了这两只恶犬，但同时也明白，今天的事怕是做不成了。身子从屋顶上跃起，几个纵跃身子已消失在黑夜中。
<br/>　　屋中，程大雷捧剑而眠，窗外的犬吠惊动了他。程大雷突然坐起，全身汗毛倒竖，匹夫剑拉开一道线。
<br/>　　“她已经来了。”
<br/>　　第二天，阿喜立在程大雷房内，头恨不得耷拉到地上。
<br/>　　“大当家，我已经带人查过，贼人该是从城墙爬过来的，城墙上发现被利器刺透的痕迹。然后贼人潜入了城主府。”
<br/>　　阿喜真的觉得有些无地自容的感觉，与他有同样感觉的，还有负责昨夜警戒工作的张肥。程大雷已提前提醒过他们，要加强防卫，结果还是被贼人神不知鬼不觉来到城主府。
<br/>　　如果不是那两条狗，怕对方真杀了程大雷也不会有人知道。如此说来，喜字队和张字军，还不如两条狗么。
<br/>　　“好啦好啦，我们的敌人不能以常理度之，这次的事情也不能怪你们。”程大雷安抚道：“那两只狗是怎样死的？”
<br/>　　“被瓦片砸死的，头骨被砸破，瓦片还陷在脑袋里。”
<br/>　　“啧……”程大雷叹了口气：“果然好厉害啊。”
<br/>(本章完)
