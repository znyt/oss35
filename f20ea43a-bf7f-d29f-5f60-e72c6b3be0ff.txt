        翌日。
<br/>
<br/>        一觉睡到自然醒的连星儿睁开眼后看到明司寒时，粉粉的唇瓣情不自禁地微微上扬。
<br/>
<br/>        明司寒还在沉睡中，从窗帘缝间射过来的晨阳照在他的俊脸上，纤长的睫毛在眼睑上投下一片阴影，恍若岁月静好般安恬。
<br/>
<br/>        他怎么能生得这么好看？
<br/>
<br/>        连星儿忍不住伸手摸了摸明司寒的脸，却在不经意间被他给捉住。
<br/>
<br/>        明司寒未睁开眼睛，嘴角边挂着笑容，声音慵懒中透着低沉的磁性：“我的小公主这是睡醒了吗？”
<br/>
<br/>        “嗯，你呢？”连星儿甜甜地咧嘴一笑。
<br/>
<br/>        明司寒这才睁开眼睛，看着连星儿笑颜如花的脸，忍不住抬起头来吻了吻她的额头：“公主殿下，早安。”
<br/>
<br/>        连星儿缩了缩脖子，又往明司寒的怀中钻了钻，找了个舒适的位置后，笑着说道：“这是我第一次在沙发上睡觉。”
<br/>
<br/>        也对，昨晚他们俩就这么搂在一起，躺这沙发上睡着了。
<br/>
<br/>        明司寒忍俊不禁，起身的时候，把连星儿也抱了起来：“那我送你到床上去睡。”
<br/>
<br/>        “不要！”连星儿本能地搂住明司寒的脖子，“我睡不着了。”
<br/>
<br/>        “那……”明司寒会心一笑，“让我亲一下。”
<br/>
<br/>        “好啊！”连星儿十分爽快地答应后，主动仰头凑近明司寒的俊脸送上自己的唇。
<br/>
<br/>        明司寒却没有吻上去，而是拉起连星儿的手，微微一笑：“我的公主殿下，我真的该走了，我可不想被你爹地妈咪发现，我是偷偷溜进来的。”
<br/>
<br/>        “那你打算怎么走？”连星儿不禁好奇地问。
<br/>
<br/>        明司寒吻了吻连星儿的额头，眉梢微扬：“我是怎么来的，就可以怎么离开。”
<br/>
<br/>        当连星儿站在窗户旁送走明司寒后，童筱颖刚巧推门而入。
<br/>
<br/>        “星儿，今天陪妈咪去逛街吧！”童筱颖手里端着一个长方形的礼盒走了进来，“这是妈咪上个月请服装设计大师给你量身定制的连衣裙，你穿穿看。”
<br/>
<br/>        “好！”连星儿立马乖巧地往童筱颖面前走去。
<br/>
<br/>        母女两在房间里顿时有说有笑起来。
<br/>
<br/>        另一边，明司寒离开傅夜沉的私宅后，上了一辆棕绿色的越野车。
<br/>
<br/>        驾驶座上，裴萱妍双手握着方向盘，感慨道：“看你这满面春风，就知道，你一定是把你那小女友给搞定了。”
<br/>
<br/>        “我亲自出马，能不搞定？”明司寒嘴角微扬。
<br/>
<br/>        裴萱妍不禁八卦道：“你把她‘上’了？”
<br/>
<br/>        “你觉得我会是那种乱来的男人？”明司寒一边卸下身上的“装备”，一边傅漠地瞥了裴萱妍一眼。
<br/>
<br/>        “她连星儿到底哪里好？你们都这么宠着她。”裴萱妍感叹道，就连奶奶裴冉，都很喜欢连星儿。
<br/>
<br/>        明司寒卸下自己身上的“装备”后，又端起一台笔记本电脑操作起来。
<br/>
<br/>        他得恢复傅夜沉的私宅里的安防系统，不然被傅夜沉察觉了，他的麻烦就大了。
<br/>
<br/>        裴萱妍见明司寒全神贯注地在做着他自己的事情，便没再打扰他了，只是望向了车窗外，那座像城堡一样奢华的私宅。
<br/>
<br/>        其实，每个女孩都有一个公主梦，只不过是被现实给摧残了。
<br/>
<br/>        但是她连星儿就不一样。
<br/>
<br/>        明司寒操作完后，合上笔记本电脑，对裴萱妍命令道：“开车，我们可以走了。”
<br/>
<br/>        裴萱妍这才发动了车子。
<br/>
<br/>        “你挺珍惜连星儿的哈！”裴萱妍一边开始，一边跟明司寒唠嗑。
<br/>
<br/>        明司寒双手抱臂，慵懒地靠在副驾驶座椅里，嘴角微扬：“当然，谁叫她是第一个让我动心的女孩子。”
<br/>
<br/>        “也就是说，在她之前，你都没喜欢的女生？”裴萱妍顿时感到不可思议。
<br/>
<br/>        明司寒闭上了眼睛小憩，没吭声，算是默认了。
<br/>
<br/>        昨晚，连星儿是睡好了，但是他没睡好。
<br/>
<br/>        温香玉软在怀，他只能单纯地抱着，就算有想法也不能行动，让他忍着又憋着，着实很辛苦。
<br/>
<br/>        到了他出国这天，亲朋好友都来送他了，唯独连星儿没有来。
<br/>
<br/>        张子晋、邢乐游和仲正浩他们三，一个个聒噪得不得了。
<br/>
<br/>        而明司寒站在登机入口旁，却没心思听他们的絮絮叨叨，而是在四下眺望着。
<br/>
<br/>        人来人往中，还是没有连星儿的身影。
<br/>
<br/>        其实，他知道连星儿不会来，因为起初在电话里说好了，她不会来送他，因为这不是送别，所以，她要在家里等他回来。
<br/>
<br/>        时间差不多了，明司寒和张子晋他们告别，心里有些失落地转身进了登机口。
<br/>
<br/>        在空姐的引领下，明司寒进了头等舱，刚一坐下，就有一只白净的小手，拿着手机伸到了他的眼皮子底下。
