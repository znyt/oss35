　　第106章
<br/>　　顺着刘悲手指方向望去，整座青牛山都在冰雪的覆盖下，银山玉树，天地苍莽，宛若个琉璃世界玉乾坤。
<br/>　　造物瑰伟，不过如此。
<br/>　　可总觉得其中少了些什么？
<br/>　　少了些人气。
<br/>　　现在正是吃午饭时候，蛤蟆寨已经生火造饭，炊烟袅袅。可偌大一个青牛山，也只有蛤蟆寨燃起炊烟。
<br/>　　“他们是……”
<br/>　　“断粮了。”
<br/>　　“该不会这么严重吧？”
<br/>　　“就是这么严重。”
<br/>　　情况比程大雷所想得更加严重。今年开春以来，老天没赏下一滴雨，幽州千里之地大旱，春种一石，秋收八斗。刚入冬，老天便迫不及待降下一场大雪，埋入土中的庄稼还没发芽便被冻死在土里。
<br/>　　蛤蟆寨因为系统奖励一口山泉，程大雷对这一点体会不深。蛤蟆寨的生活总之是过得去的，然而蛤蟆寨之外的生活，程大雷并无法想象。
<br/>　　千里大旱，幽州城已有人用童男童女祭天秋雨；北蛮部草原便沙地，牛羊成群饿死，呼尔勒带人来黑石城打草谷；刘悲的家乡有恶霸逼死交不出田租的佃户，刘悲怒而杀人，带着关张前来投奔程大雷……
<br/>　　诸多事情，千头万绪，但追本溯源其实都是由这场大旱引起。
<br/>　　食不果腹，饿殍满野，人们易子而食，析骨而炊这些史书上才能看见的文字，距离程大雷也并不遥远。
<br/>　　“整座青牛山都快吃不上饭，再如此下去，他们肯定会再次铤而走险。到时五千饿夫，为了活命，怕蛤蟆寨未必能挡得住。这件事，大当家要早作打算。”
<br/>　　程大雷已可以想见这些人为了活命，不顾一切向蛤蟆寨冲锋的景象。凭着蛤蟆寨百八十口人，怕还真挡不住。
<br/>　　“不过这也未必不是个机会，如今这年景，人命贱如狗，我们正好趁这个机会招兵买马，扫平青牛山其他山寨，一统青牛山。”刘悲。
<br/>　　程大雷没有回答，依旧看着山下。只见远方山林中，有两个人鬼鬼祟祟，正往雪中埋着什么。
<br/>　　“他们在干嘛？”
<br/>　　“埋死人。”
<br/>　　“埋个死人用偷偷摸摸的么？”
<br/>　　“不偷偷摸摸被人挖出来吃了怎么办。”刘悲理所应当道。
<br/>　　程大雷胃里一阵干呕。
<br/>　　“已经开始死人了，再这样下去，死的人会越来越多。”刘悲道：“大当家如果看不过去，可以把我们山寨的粮食散出去，只是这样一来，我们山寨怕也熬不过这个冬天。”
<br/>　　刘悲看向程大雷：“两条路，大当家会怎么选？”
<br/>　　“似乎第二条路对我没什么好处？”程大雷。
<br/>　　“也不能说一点好处没有吧，至少能得人心，或者求个心安。”
<br/>　　程大雷沉默下来，一时没有做声。
<br/>　　嘟，正所谓霸道无情，强者争霸天下的路，注定是用无数尸骨堆砌，趁此机会招兵买马，扫平青牛山，勇敢迈步你崛起的第一步。
<br/>　　嘟，正义的少年喔，你周围的人们正在水深火热中，身为一个心怀正义的少年，尽己所能帮他们度过难关。
<br/>　　正在程大雷沉默的光景，他接连收到两条系统提示。这还是系统同时发布两个任务，一道选择题，当程大雷接受其一的时候，必须放弃另外一个。
<br/>　　这不太好选。
<br/>　　午饭时程大雷没有吃多少东西，蛤蟆寨的伙食还算不错，有酒有肉，可捧着手中的碗，程大雷却有种食不下咽的感觉。
<br/>　　下午程大雷独自一人离开山寨，他牵上瘦马，腰上佩剑，单人独骑去了落叶城。
<br/>　　随着时间流逝，落叶城通缉他的风声已落了下来。可就算这样，程大雷也是在塞给卫兵一袋铜板后，才顺利混入落叶城。
<br/>　　牵马走在街上，街道冷清得紧，天寒地冻的也没多少人会出门。
<br/>　　程大雷是来寻苏樱的。
<br/>　　他牵马走进酒楼后院，早有伙计把马接过来，酒楼大半都是蛤蟆寨的人，所以都认识程大雷。
<br/>　　“大当家。”
<br/>　　“嗯，苏小姐呢？”
<br/>　　“老板娘在大厅，今天恰好有人找他？”
<br/>　　“什么人呐？”程大雷信步向大厅走去。
<br/>　　“苏家的人。”
<br/>　　酒楼大厅，散座着几个人，小蝶站在苏樱身后，紧张得心脏砰砰直跳。
<br/>　　今天苏家来的人不少，都是苏家的老人，还有苏樱的同族兄弟，以及城里几位有威望的老人。
<br/>　　苏墨林是苏樱的大伯，在家中威望很高。他从进门开始便黑着一张脸，斜睨着周围。
<br/>　　“苏家的女儿在外面抛头露面，成何体统，苏家的脸面都被你丢尽了。”
<br/>　　“我已被赶出家门，从那一刻起，我便不再是苏家的人。”苏樱。
<br/>　　“这酒楼本来就是苏家的，你既然自认不是苏家的人，便该把这酒楼还回来。”
<br/>　　“喂，你们这是不讲道理，当初是你们把酒楼留给小姐，现在看酒楼生意好了，便想把这酒楼要回去。”小蝶。
<br/>　　“这里有你说话的资格么，果然是有怎样的主子，就有怎样的奴才。”苏墨林冷哼一声，望着苏樱道：“现在要么把这酒楼还回来，从此你的生死和苏家无关。要么就乖乖和我回家，别在外面丢苏家的脸。”
<br/>　　“这里干嘛呢？”
<br/>　　“是谁在说话，这里就没一个懂规矩的人么？”苏墨林斜过眼去。
<br/>　　一个裹着大袍子的人挑开门帘走进来，抬起眉毛：“我在说话，怎么了？”
<br/>　　眼中精芒暴露，杀机淋漓，苏墨林宛若被山中的大虫盯上一般，一股凉气顺着脊椎骨往上钻。
<br/>　　黑石城外，斧劈呼尔勒，血不仅落在他身上，也浸入他心里。耳濡目染，程大雷早沾染一股豪横之气。
<br/>　　“你是谁？”苏墨林下意识竟站起来，如临大敌。
<br/>　　“滚！”
<br/>　　山林中的野兽闯入猪圈，哪怕只是打个喷嚏，就已能令院内的家禽牲畜瑟瑟发抖。
<br/>　　小蝶惊讶的看到，刚才还如岳临渊的苏家人，此刻竟灰溜溜的离开。
<br/>　　苏樱目光落在程大雷身上，上上下下打量一遍，从程大雷腰上扯下一块手帕。
<br/>　　程大雷一身冷汗：“哈，我正要送给你，没想到就被你发现了，你真聪明。”
<br/>　　“你怎么来了？”苏樱不置可否。
<br/>　　程大雷耸耸肩：“我来找你买点粮。”
<br/>　　
<br/>(本章完)
