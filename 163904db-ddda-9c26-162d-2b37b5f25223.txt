　　“诛仙剑阵，戮仙弑神，绝灭一切，转！”
<br/>　　魔祖罗睺大喝一声，控制着诛仙剑阵核心，阵法变幻无穷，杀机四伏，让人分不清楚对错，不好下手。
<br/>　　轰隆隆！
<br/>　　四柄凶剑突然间释放出无尽杀气，这杀气与一般红色杀气不同，它是纯黑色的，散发出一种破灭一切，灭尽天地，屠尽天下又何妨的意志。森寒的杀气凝聚成一朵朵冰冷的黑色冰花，飘散在天地之间，花中，是无尽杀机。
<br/>　　“轰隆……轰隆……轰隆……！”
<br/>　　变化还没有停止，天地间同时又出现四道惊天剑柱，剑柱擎天，散发着无尽锐利杀机，剑气飞扬，剑意通天，一道道剑气纵横虚空，将虚空撕开一道有一道裂缝。
<br/>　　同时，罗睺头顶飞出一座黑色莲台，护住自己，正是四大莲台之一的十二品灭世黑莲，为先天极品灵宝，内含毁灭法则，防御无双。
<br/>　　同时，他的手中更是出现一柄青色木制长枪。长枪古朴无华，道韵自然，没有绝世杀机，亦没有盖世神威，更没有灭世道法，有的只是普普通通，平平凡凡。
<br/>　　要是放在地上，它会毫不起眼，就跟一般的木头没什么两样。但，它如今在魔祖罗睺手中，又怎会平凡？
<br/>　　它就是三十六品混沌青莲莲茎所化的先天至宝弑神枪！
<br/>　　弑神枪虽为五大先天至宝之一，但它并没有其他四大先天至宝那般出名，原因是因为后面四大先天至宝的主人后来不是天地圣人就是天庭妖皇，它们想不出名都不可能。
<br/>　　而弑神枪经此一战之后，后来却没有什么出奇的战绩，故而，它不怎么出名。但，作为先天至宝，不能因为它不出名就小看它。
<br/>　　弑神枪，弑神，弑神，专门破人神识，灭人元神，杀伤力极为惊人，让人防不胜防，甚至连圣人都能伤到其元神，可见其威力惊人。
<br/>　　“杀！”
<br/>　　用诛仙剑阵缠住四人之后，罗睺取出先天灵宝，就向着穹天老祖杀去。毕竟，诛仙剑阵也不是久战之法。现如今四人破阵，他虽然嘴里说是不屑，但实际上他心里早已警惕万分了。只有用剑阵将他们分开，之后再逐一攻击，方能灭杀他们。
<br/>　　要不然，真是四人一同围攻他一人，他还真不好对付，能修炼到如今之人，会简单吗？
<br/>　　“杀！”
<br/>　　罗睺手持弑神长枪，一枪毫不保留地刺向正在破除绝仙阵门的穹天老祖。长枪无光，朴实无华，刺破虚空，没有发出一丁点声音，就这样悄无声息的穿过虚空，刺向穹天老祖。
<br/>　　“擎天一棍！”
<br/>　　穹天老祖的法宝是一根灰色长棍，为上品先天灵宝，有擎天之势，威力不俗。配合上穹天老祖的穹天战法，更是厉害非常。
<br/>　　“噗嗤”
<br/>　　诡异的是，穹天老祖的长棍并没有预想中的与罗睺的长枪交接，发出巨大碰撞之声，而是罗睺的长枪诡异的透过虚空，避开长棍，刺入穹天老祖脑袋之中，溅出朵朵血花。
<br/>　　而此时穹天老祖的长棍却打在了罗睺头顶的灭世黑莲之上，发出巨大声响，但罗睺防御无双，只是被震得倒退几步，并没有受到多大伤害。
<br/>　　穹天老祖双眼呆滞，还保留几分震惊，口中却轻声问道：
<br/>　　“为什么？”
<br/>　　“因为你不了解本尊的法宝，而我却对你了如指掌。本尊法宝先天至宝弑神枪，能穿破虚空，灭人神魂。”
<br/>　　“原来如此！”
<br/>　　是啊，原来如此，自己的底牌对方已经完全了解了，而自己对于对方却毫无所知，自己已经大意了，自认为四人一起对付一个还没有成圣的存在，心里就开始有了松懈。自己这样还不倒霉还有谁倒霉啊？
<br/>　　永远不能小看任何一个人，更何况此人不是一般人。
<br/>　　“砰！”
<br/>　　穹天老祖倒地，至此，仅仅一击，围攻魔祖罗睺的四大老祖就陨落一个。
<br/>　　收起长枪，罗睺不再停留，直奔乾坤老祖而去。
<br/>　　“乾坤老贼，你给我死来！”
<br/>　　罗睺大叫一声，声音传遍八方，带着熊熊杀气！
<br/>　　目标，直指乾坤老祖！
<br/>　　“轰隆隆！”
<br/>　　乾坤老祖头顶极品先天灵宝乾坤鼎，手持乾坤尺正在竭力抵抗戮仙剑气。
<br/>　　在鸿钧四人之中，除了已死的穹天老祖，就是他的实力最弱了。
<br/>　　乾坤鼎虽然是一件极品先天灵宝，然，功效单一，只是能够返后天为先天，内蕴一个小世界，没有什么攻击力，防御力也是一般。
<br/>　　故而，此次在诛仙剑阵之中，他的形势有些艰难，只能勉强抵挡剑气，极为狼狈。幸好他还有一件具有乾坤法则的攻击灵宝乾坤尺，只不过乾坤尺仅仅只是一件中品先天灵宝，攻击力也不算太强。
<br/>　　他乾坤老祖修炼的是乾坤道，乾坤道是世界演化之道，并不以攻击力著称。
<br/>　　“不好，不……”
<br/>　　陡然间，乾坤老祖身躯巨震，感受到一股极为恐怖的杀机锁定自己，让自己根本无处可逃，不由惊恐大叫一声。
<br/>　　“死！”
<br/>　　罗睺森寒的声音传入他的耳中，乾坤老祖便见到罗睺化作一道黑光，提着弑神枪刺向自己。罗睺那阴冷中蕴含无尽杀机的声音，令乾坤老祖不禁打了一个冷颤，他只感觉自己浑身汗毛根根竖立，一股生死危机笼罩在自己心头。
<br/>　　“啊！”
<br/>　　罗睺狞笑之中，乾坤老祖惨然大叫。罗睺几乎拼尽全力的偷袭之下，配合戮仙剑门的死亡之力的攻击，竟然一举偷袭成功，打落乾坤鼎，在乾坤老祖祭出乾坤尺之前一击刺中乾坤老祖，弑神枪没入心口之中。
<br/>　　“死！”
<br/>　　罗睺大喊一声，同时，手中弑神枪狠狠地一震，庞大的力量爆发，顿时毁灭了乾坤老祖的肉体。弑神枪的枪意向着乾坤老祖元神之处冲去势要将他元神灭杀。
<br/>　　乾坤老祖元神一纵，刚想遁走，寻求一线生机。
<br/>　　“不！”
<br/>　　可惜，罗睺的弑神枪可不是那么好惹的，霸道的弑神之意，追向乾坤老祖元神，将他震碎，同时，枪身上传来一阵恐怖的吞噬之力，彻底将他那破碎的元神强行扯入，炼化成弑神枪的威力。
