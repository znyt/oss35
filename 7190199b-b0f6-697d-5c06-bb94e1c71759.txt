
在王惊蛰这类人的眼里，长相是不分好坏的，郭德江的女儿虽然长得挺让人不寒而栗但他还真没太在意，只说对方这受不起的面相，看在他眼里却挺有意思的。<br/>　　<br/>　　什么叫受不起面相呢，这算是行话了，笼统的解释就是这人的脸和她的身份不相符，而且还得是后天营造出来的，和先天面相还不太一样。<br/>　　<br/>　　这郭德江女儿呢就是此面相了，紫气压头，颧骨颇高什么的，比富贵脸还要跋扈，这是妥妥的女帝面，只是却没有成型，也成不了，因为她的身份，命格都不匹配，全天下有女帝王相的人只有一个，那就是武则天，无论是慈禧老妖后还是孝庄太后都算不上。<br/>　　<br/>　　所以，郭德江的女儿苦逼了，由于她撑不起来这面相，那自然就是麻烦缠身了，那要说以前为啥没问题呢，那是因为今年她刚好二十四岁，到了本命年，正好应景了，琐碎事多，挺过了今年的话这孩子还得一路衰败下去，最后搞不好都得把郭家的产业给带偏了。<br/>　　<br/>　　看着王惊蛰一时半会都没说话，郭德江就有点担忧的问道：“她这是……”<br/>　　<br/>　　王惊蛰都有点直挠头，问题是找到了，但是原因在哪他也有点懵，这种状况其实挺罕见的。<br/>　　<br/>　　郭德江女儿用那只肥硕的舌头舔着肉嘟嘟的手指上的油渍，仰头说道：“爸，他是干嘛的啊？盯着我都不动了，你看他的眼神，隐隐还有爱慕之情，这是你给我介绍的男朋友，送我的生日礼物么？这人，丑是丑了点，但也能凑合着用吧”<br/>　　<br/>　　王惊蛰在心里“哎呀卧槽”了一声，我他么还没说你长得像霸王龙似的，你倒是先说我丑了？<br/>　　<br/>　　郭德江尴尬的说道：“王先生不好意思，这孩子被我惯坏了，您见谅”<br/>　　<br/>　　王惊蛰不以为意的摆了摆手，随即也没说话，背着手走出客厅在外面转了一圈，转完之后又再进来，楼上下逛了两遍，这宅院的风水是没错的。<br/>　　<br/>　　“你家里供了什么神位没有？”王惊蛰皱眉问道。<br/>　　<br/>　　“以前做生意的时候供过关公，后来认识位朋友，说我生意平稳了后再供关二爷就不太合适了，然后被他给送走了，从那以后我就什么也没供过了，只是逢年过节的时候会去庙里捐一些香火钱”<br/>　　<br/>　　“把她的生辰八字报给我……”<br/>　　<br/>　　郭德江报了女儿的八字，王惊蛰略微掐指一算，发现霸王龙的八字还算行吧，只是略显柔弱了一些，算是水命，比较容易生病，破财，招灾，只不过郭德江的气运比较强横，所以能压住他女儿的倒霉相。<br/>　　<br/>　　算到这，王惊蛰依旧没算出来霸王龙的后天女帝相是怎么出来的，因为这种纯粹是营造出来的面相，会由很多原因造成的，你不知道是哪个因素，就很难判断出根在哪里。<br/>　　<br/>　　霸王龙听着王惊蛰和郭德江的对话，就眨了眨一对肿眼泡，眼睛有点发亮的问道：“爸，这是你找的算命还是看风水的先生啊？”<br/>　　<br/>　　“看事的”<br/>　　<br/>　　霸王龙顿时伸出一只肥嘟嘟的手爪子，伸到王惊蛰面前，说道：“师傅你给我看看姻缘吧？我肿么感觉自己有种要桃花盛开的意思了呢，心头有如小鹿一般撞啊撞的，你快给我看看，我是不是要处对象了？”<br/>　　<br/>　　王惊蛰瞥了眼对方的肉爪子挺无语的看着郭德江，这姑娘的手胖的都快要看不出纹路来了。<br/>　　<br/>　　郭德江绷着脸，呵斥道：“小赵，你给我消停点，我是和王先生谈正事呢，你把嘴闭上”<br/>　　<br/>　　霸王龙撇着厚厚的嘴唇子，转过脑袋哼了一声，又缩在了沙发上。<br/>　　<br/>　　“唰”王惊蛰脑袋里忽然有个念头一闪而过。<br/>　　<br/>　　“你刚才叫她什么？”王惊蛰忽然问道。<br/>　　<br/>　　郭德江愣了愣，看了眼女儿说道：“叫她，小赵啊”<br/>　　<br/>　　“小赵，哪个赵？”王惊蛰皱眉问道。<br/>　　<br/>　　“就是那个，上面是日月，下面是个空的曌啊”<br/>　　<br/>　　王惊蛰当即倒吸了口气，点头说道：“曌，日月为明当空曌……她叫郭曌？”<br/>　　<br/>　　“对！”<br/>　　<br/>　　王惊蛰闭着眼睛无语的叹了口气，说道：“这个名字是谁给他起的”<br/>　　<br/>　　“请个测字的先生，当年这孩子出生以后，我就找人为她起名，对方问我想希望孩子以后怎么样，我就说希望她能继承我的事业越做越大，成为一个商业帝国来那就最好了，这起名的先生就说那就叫郭曌吧，这个曌是帝王名，是个好兆头”郭德江说到这就意识到了，接着问道：“孩子的名，不对？”<br/>　　<br/>　　王惊蛰冷笑道：“下次请人眼睛擦亮一点，你找的这个人连个半吊子都不算……”<br/>　　<br/>　　当年武则天还没有称帝的时候，手下有个大臣叫宗秦客，为武则天献上了一字就是曌，武则天以前名为武照后来才改为了武曌，这个曌字就是日月凌空普天同照的意思，但还有一层意思就是，日月当空中的空为天，这是把天都给压在了下面，所以天下间也就一代女王武则天能用得了这个曌字，剩下在有人用的话，特别是女人，谁取这名谁倒霉。<br/>　　<br/>　　也有人说我也碰到过名字里有曌的人，她怎么就没事呢，这还得跟自己的八字有关，要是这人八字很硬，性格极强的话，这曌字也能受得起，反之郭德江女儿的八字就偏软了一些，再起了这个名字，又赶上了今年的本命年，自然而然的就成了倒霉催的了。<br/>　　<br/>　　所以，名字不能乱取，不是什么好的喻义都可以用在自己身上的，你得量力而为才行。
