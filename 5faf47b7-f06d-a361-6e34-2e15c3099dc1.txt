　　第379章
<br/>　　若程大雷有防备，樊梨花也伤不着他。可如此夜色如此人，程大雷也有些想入非非，难免放松了警惕。
<br/>　　这一下摔得结结实实，程大雷感觉自己全身骨骼都要散架，塌在地上一时片刻站不起来。
<br/>　　草原儿女，敢爱敢恨，精通的不仅骑术，还有……摔跤。
<br/>　　程大雷揉着发酸的肩膀，下意识去勾樊梨花的手掌。樊梨花此刻也有些不清楚，昏昏沉沉，就感觉手掌被人抓住。几乎是下意识动作，她反手抓住对方手腕，身体如山倒下来，一计肘击，撞在程大雷小腹。
<br/>　　我擦……
<br/>　　程大雷感觉肚子里翻江倒海，刚才吃的东西，差点都吐了出来。
<br/>　　这姑娘手真黑，是要谋杀亲夫呐。程大雷想着不能任她如此放肆，一个鲤鱼打挺从地上跃起，想要先制住樊梨花。
<br/>　　几乎是他刚刚摆好架门，樊梨花的拳头便砸来。程大雷抄手接住，借力使力，将她凌空举起。终究不敢使全力，高高抬起，轻轻落下。
<br/>　　樊梨花对他可是出手不留情，今日借着酒劲，过往的恨意都泛了上来。可程大雷却是畏首畏尾，只是仗着身体灵便躲闪。
<br/>　　程大雷恍然意识到一种情况，有人喝醉了倒头就睡，可有人喝醉了却是会耍酒疯的。程大雷本以为樊梨花是前者，但谁想到她竟然是后者。
<br/>　　二人一个打，一个躲，本来精致的摆设，稀里哗啦碎了一片。桌子也歪了，花瓶也砸了，床底下的酒坛子稀里哗啦被打破，酒香扑鼻而出，氤氲满整个房间。
<br/>　　樊梨花只感觉身在云里雾里，酒香冲入鼻孔，脸上挂出一抹诡异的笑容。
<br/>　　笑得程大雷浑身冷汗，这姑娘该不会喝酒烧坏了脑袋吧。他试探着靠近，小心翼翼把手搭在对方肩上，还没来得及开口，樊梨花就双手抓住他的手腕，背过身去，身体发力。
<br/>　　又来！
<br/>　　还是同样的招数，程大雷如果再被摔在地上，他在江湖上也没必要再混了。
<br/>　　他双足立地，一把就将樊梨花揽在怀里，抱着她双脚离地。樊梨花又急又羞，脑袋也是时而清醒，时而糊涂。只感觉一身燥意往上涌，浑身上下着火一样发烫。
<br/>　　温玉暖香抱在怀里，酒香和体香掺在一起，沁鼻幽兰，程大雷也有种如梦似幻的感觉。
<br/>　　“喂，疼……”樊梨花下意识道了一声。
<br/>　　程大雷这才意识到，自己胳膊勒住了樊梨花的胸脯，樊梨花的身体果然是……健康呐。
<br/>　　他下意识松开，樊梨花如一条游鱼般钻出他的怀抱，滑不留手，一脚反踹向他裆部。
<br/>　　程大雷倒抽一口凉气，这完全是置自己下半生幸福生活于不顾。
<br/>　　他托住樊梨花脚踝，往后轻轻一拽，托着她往后倒退一步。樊梨花如此撒泼，程大雷已有几分不耐烦，关键是打了这么久，程大雷的酒意也泛了上来。
<br/>　　他推着樊梨花的脚踝将她摁在墙上，扯了一条床单，将她身子绑住。樊梨花再如何发力，也挣脱不开舒服。
<br/>　　而她也察觉到，这个姿势有些不雅，挣扎骂着。
<br/>　　“混蛋，你放开我！”
<br/>　　程大雷晃了晃脑袋，想要将醉意驱赶出去，可越晃脑袋越沉的厉害。目光落在樊梨花身上，一时怔住，心尖有些发颤。
<br/>　　樊梨花此刻倒在床上，头和脚绑在一起，不得不说身体柔韧性是相当不错。眼角挂着泪痕，脸上的表情又是怒，又是羞，又是怕，这匹暴烈的马驹，此刻像是被剥了皮的羊羔一般楚楚可怜。
<br/>　　程大雷心中一荡，鬼使神差的倒在樊梨花身边，将她揽在怀里。樊梨花只感觉一股燥意，全身滚烫，程大雷的手肆无忌惮的顺着脚踝摩挲上来，心中隐隐明白要发生什么，却好像又不明白。随着一件件衣服解开，樊梨花身子软得可怕，心脏砰砰直跳，如同打鼓一般。
<br/>　　只感觉被程大雷蒙住了脸，呼吸变得越来越炙热，她心里也明白将发生一件大事，其实怕得要命。
<br/>　　模糊抓住了程大雷的手掌，十指紧扣。随着一身从心令伸出发出的尖叫，她忽然发力，掰断了程大雷的手掌。
<br/>　　……
<br/>　　夜尽天明，是天将要亮还未亮的时候，屋子里油灯的火焰轻轻晃动。
<br/>　　樊梨花用被子拥着身体，露在空气中的双肩表示她的身体一丝不挂，她脸颊泛红，看着地上的程大雷，开口不是，不开口也不是。
<br/>　　程大雷蹲在地上，捧着左手大拇指，不停在上面吹着凉气。
<br/>　　十指连心，更何况是拇指活生生被折断。樊梨花也能理解程大雷目前的感受，尴尬道：“我第一次，没什么经验，弄痛你了吧。”
<br/>　　程大雷抬头白了她一眼，咦，这话怎么听怎么感觉不对。
<br/>　　“要不我帮你看看？”樊梨花小心问。
<br/>　　“啊，不必不必，我自己能够处理，小伤小伤。”程大雷急忙摆手，他可不希望再次负伤。
<br/>　　程大雷给自己做了简易正骨，过程中疼地龇牙咧嘴。他悄悄抬眼看着樊梨花，发现樊梨花也正偷偷看着他。
<br/>　　二人目光相对，樊梨花忙将头低了下去，两眼水汪汪的，如同蕴着秋波。
<br/>　　嘟，樊梨花成为压寨夫人，触发条件达成，攻城模式开启。
<br/>　　正在程大雷处理好伤口，想入非非之际，脑海中收到一份系统提示。
<br/>　　原先到达66%的进度条消失，显然已经全部完成。
<br/>　　程大雷有些无语，为何这进度条完成，要和压寨夫人有关。系统一路的任务指引，究竟打算让自己成为怎样的人。
<br/>　　山贼之心，恐惧值可以兑换装备，压寨夫人任务、隐藏的邪恶值、召唤出来的和珅……这一切令程大雷觉得，系统的最终目的，会不会是让自己成为那种：十恶不赦，卑鄙狡诈，贪财好色，正义之士人人得而杀之的恶贼。
<br/>　　自己拿到的剧本，究竟是主角，还是反派大BOSS……
<br/>　　队长别开枪，我是好人呐。
<br/>　　程大雷从内心深处发出一声呐喊，然后随着一条系统提示，他收到了系统奖励。
