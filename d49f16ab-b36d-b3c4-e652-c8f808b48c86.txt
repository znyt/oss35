<br/>    紫霞仙子一定要跟着，陈明也没有什么办法，反正他和紫霞仙子实力差距不是特别大，问题不大。
<br/>    
<br/>        一路前行，紫霞仙子道：“你知道吗，传闻，还魂水是一种美丽之极的生灵的泪珠，她每流一滴眼泪，便是一滴还魂水。”
<br/>    
<br/>        陈明摇头。
<br/>    
<br/>        也就女人会信这种鬼话了！
<br/>    
<br/>        继续向下，在前方，忽然出现了一具巨大的尸骨，随便一根骨头，都有着一座房屋那么大。
<br/>    
<br/>        紫霞仙子道：“根据雪山宗所说在，在这里，这种巨大的尸骨，在脑骨之中可能就孕育得有还魂水。传闻这种巨大的尸骨便是传说中的龙鲲的尸骨，龙鲲一生吃下了无数的生灵，而那些生灵的灵魂，也被封锁在龙鲲的身体之中，最后凝结成还魂水。”
<br/>    
<br/>        陈明听到这话，哪里还等的了，只是道了一句，“坐稳了！”
<br/>    
<br/>        四只小爪子舞动的飞快，如同一根利箭瞬间冲进那巨大的头骨之中，即便是以卓青瑶的修为，还是差点仰倒在陈明的身上。
<br/>    
<br/>        在眼中忽然出现一抹晶莹之色，如同水晶般璀璨，陈明的眼中一亮，这就是还魂水吗！
<br/>    
<br/>        刚想伸出爪子直接拿到手里，身旁却是有着一道倩影掠过，直接在陈明的身前出现，伸手将那一抹晶莹之色收入手中。
<br/>    
<br/>        陈明问道：“怎么样，有几滴还魂水？”
<br/>    
<br/>        紫霞仙子摊开素白的小手，之中有着六滴还魂水正乖乖的躺在手中。
<br/>    
<br/>        紫霞仙子道：“这一次大家都没怎么出力，见者有份，一人两滴吧。”
<br/>    
<br/>        陈明心说：虽然凶巴巴的，但是还挺公道的。
<br/>    
<br/>        陈明将四滴还魂水收下，然后直接丢了一滴到嘴里，闭上双眼，数十个道宫瞬间开动，不过十息之间，陈明已经是彻底吸收了这一滴还魂水，睁开双眼，心念确实强大了那么一丢丢，但是强大的不多啊！
<br/>    
<br/>        估摸着，能多操控十分之一柄飞剑。
<br/>    
<br/>        这进来已经七八天了，才弄到这么两滴，自己要想真的操控万剑，这得到什么时候啊！
<br/>    
<br/>        至少得喝一缸还魂水吧？
<br/>    
<br/>        等到弄到一缸还魂水，怕不是外面都已经大家愉快的手牵手去死完了！
<br/>    
<br/>        至于说什么奇遇遇到一缸还魂水，这不是扯淡吗？
<br/>    
<br/>        看了一眼自己的光环，并没有幸运光环，我他喵的又不是主角，这种撞大运的做法怕不是这辈子都没戏了！得想个其他的办法才行。
<br/>    
<br/>        仔细回想了一番炼化还魂水的感觉，其中有着很大一部分逸散掉了，还有一部分没有彻底吸收，而且没有彻底激发。
<br/>    
<br/>        等等，像是这种情况，是不是可以炼丹啊！
<br/>    
<br/>        还魂水也是材料吗，有什么不行的！
<br/>    
<br/>        恰好自己储物戒里有着古仙门最近给的小山般的药材，灵药也有着一大堆，未尝不可以试试看啊！
<br/>    
<br/>        紫霞仙子道：“你怎么愣住了？”
<br/>    
<br/>        陈明道：“我忽然有一个大胆的想法，这还魂水，是不是也可以炼制成丹药呢？就好像普通的药材，炼制成丹药，可以有着壮大气血的效果，将原来的药材效果放大了数倍，如果说，这还魂水也能炼制成丹药的话，那么，我们完全可以让其他的人王在我们这里用还魂水换丹药啊！这样一来，就相当于所有的人王，都得将他们一部分的还魂水分给我一份，这样一来，我的心念就可以足够强大了！”
<br/>    
<br/>        紫霞仙子：.......
<br/>    
<br/>        还有这种操作？
<br/>    
<br/>        炼丹师果然都不是正常人，按照紫霞仙子的想法，不就是大家凭运气在这里寻招还魂水和宝物吗？然后遇到比较多的还魂水，如果都发现了，大打一场，多有修仙争夺机缘，与天争命的感觉，怎么到了陈明这里，画风突然就不对了！
<br/>    
<br/>        这家伙满脑子都是怎么从其他人的身上刮下一层皮来！
<br/>    
<br/>        陈明道：“嗯，有道理，伟大的领袖曾经说过，我们要发挥群众的力量，团结，才是力量！容我回想一下这还魂水的效果，药性，药理，分析一下如何将这还魂水炼制成丹药。”
<br/>    
<br/>        陈明趴在头骨上，开始仔细思考着这个问题，在脑海中分析了一番之后，伸手摸出几百块灵石，直接在这个头骨之中布置出一个隔开水流的阵法，头骨之中的水流被清除干净，陈明让卓青瑶下来，然后站在干燥的头骨之中，拿出一个丹炉来，然后挥手打出一道火焰，落在丹炉之下。
<br/>    
<br/>        紫霞仙子对于陈明直接挥手打出一道火焰来也是有点奇怪，你本体不是一只乌龟吗？怎么打出火焰来了的？
<br/>    
<br/>        不过紫霞仙子也没有多问，修行界奇怪的事情多了去了，把陈明当成一个炼丹师，问题就不大了。
<br/>    
<br/>        陈明挥手，从储物戒之中再次取出一张桌子，然后将各种瓶瓶罐罐放在桌子上，挥手开始准备材料，“按照还魂水的药性，应该需要一百三十八种药材，其中灵药二十三种，而且不能用火炼，这还魂水本就是极寒之物，遇到火焰就消散了，必须先经过水炼，然后水火合一，阴阳和合，这才符合天地大道！看来我还得布置一个阵法！”
<br/>    
<br/>        陈明挥手，两柄国器落下，一火一水，阴阳两极大阵陈明可是见过的，应该可以用。
<br/>    
<br/>        陈明手中握着万道天竹，拂尘挥动，一个太极阵落于脚下，陈明脚踏八卦，拂尘一扫，手中的国器一柄接着一柄落下，分别立于八卦的方位，踏入大阵之中，陈明瞬间进入天人合一的状态，这种状态下，让对于周围的一切可谓是绝对掌控，不可能出一丝差错！
<br/>    
<br/>        紫霞仙子愣愣地看着陈明布阵，脚踏八卦，我没有看错吧，这炼丹还需要布阵的？！天人合一的境界，她也达到了，对于陈明现在的状态，她再清楚不过，这是天人合一，她这辈子长这么大，还没见过炼丹炼到天人合一的，这是多丧心病狂！
<br/>    
<br/>        而且，还是一个入道境！
<br/>    
<br/>        看着陈明稳健的步伐，陈明已经开始踏九宫了，“九宫鸳鸯，四才无极，阳火最盛，应该在这里！”
<br/>    
<br/>        陈明道完，挥手将原来的丹炉直接放在了这里，而这里，便是太极阴阳鱼黑鱼之中的那一点，本是无火的地方，地面之上，忽然升起一堆炽烈的火焰来！
<br/>    
<br/>        紫霞仙子看着无意间看到陈明的双眼，坚毅而自信，原来这个不着调的家伙，认真起来还是蛮有意思的，不知道他那面具下是什么样子。
