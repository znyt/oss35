
    三月底的蓉城沐浴在野花馥郁的芬芳中，边城路的落叶乔木换上了葱绿的颜色。<br/>　　<br/>　　    在倒映着街道和绿木一尘不染落地玻璃窗的天行道馆内。<br/>　　<br/>　　    蒋舟，还有蒋舟现任女朋友也是作为天行道馆会计的蒋艳，谢飞白和他那建设天行道馆二层楼的马宏宇，胡睿，卢玲，邓维等人，先后和办公室的程燃见了面。<br/>　　<br/>　　    蒋舟在结算时对程燃兴奋的道，“天行道馆二层楼在二月初开业以来，已经差不多两个月时间，当初开设天行道馆，装修花了30万，设备100万，你让伏龙开发了一套管理软件系统，杂七杂八的加起来，开业的成本就达到了140万。我还一度担心这个钱收不回来，但是没想到，我们二层楼的两个月下来，计时收入就达到了30万，而会员卡的充值收入则是90多万！”<br/>　　<br/>　　    蒋艳快速的补充道，“会员卡销售这里面主要还是第一个月比较多，头月达到了60万。”<br/>　　<br/>　　    蒋艳是蒋舟一个学校的，本身在经贸院就是会计专业，不知怎么就和蒋舟谈起了朋友，二层楼开起来过后，蒋舟跟程燃提了一下，规模扩大了，他需要帮手，程燃也就同意了他把蒋艳提来作帮手，作为天行道馆的会计。<br/>　　<br/>　　    天行道馆是会员制度，每个人消费都需要办会员卡，通过先充值后结账的方式。当然首充会有奖励，比如充十块算十二元，充二十算二十五，以此递增，100，200……500，1000，2000，充值最高的5000元可以获赠899豪华大礼。<br/>　　<br/>　　    这里面就有个附近理工大学的老教授一口气办了一张卡，结果是给自己孙女办的，一方面他孙女看上了899豪华大礼可选择的一张小提琴，其次他看中的是学习区的氛围，希望自己孙女在这边学会电脑基础操作，而在家里买一台电脑堆着占地方没学习地方不说，投入更大。<br/>　　<br/>　　    禁止不满十六岁入网吧的规定在三年后才会实施，当然这也并不代表着老教授就愿意让自己孙女进入一般性的网吧，而是看自己孙女经常偷偷去网吧，这才选了天行道馆这么一个正规环境好的保险地方。<br/>　　<br/>　　    堵不如疏嘛。<br/>　　<br/>　　    计时收入其实是包含在会员卡充值收入之中的，会员卡充值收入其实才算是二层楼的真正收入，因为会员卡的充值金额不会退返。同一张会员卡只能转让，在你享受了充值优惠后，也就默认放弃了退返的权益。<br/>　　<br/>　　    会员卡的存在又进一步巩固消费人群的忠诚度，当然这个收入会逐步放缓，但不会低于每月的计费收入，而且在二层楼往后的一些促销和庆祝比赛活动中，又会造就一波又一波的高峰。<br/>　　<br/>　　    “扣除我们的成本，净利润大约在85万。”<br/>　　<br/>　　    “除此之外，这两个月天行道馆一层楼的净利润在24万。”<br/>　　<br/>　　    光是天行道馆，两个月的收入就能达到一百多万。<br/>　　<br/>　　    三个月就相当于二层楼回本了。<br/>　　<br/>　　    这果然是一个信息社会爆发新世代啊。<br/>　　<br/>　　    网吧这种东西程燃其实以前压根没有接触和实际操作过，如今一步一步走到眼下，眼看着一二层楼都进入正轨，真是让人感到一种满足感。<br/>　　<br/>　　    程燃仔细的算了一下，他现在现金奶牛就有两头，桌游方面每个月能够卖出**万套，综合算下来一个月销售收入三十多万，和程齐对半平分后，一个月他收入十几万现金。如今天行道馆又强势起来，抛除开业的群聚效应，未来可能一个月收入稳定在三十到五十万之间。<br/>　　<br/>　　    天行道馆两层楼建设起来后，完全超越桌游的收益，成为了他目前最为强势的现金源产出！<br/>　　<br/>　　    真金白银啊！<br/>　　<br/>　　    这个时候的一个月几十万什么概念。他重生来之前的那个时期，就有说法十年前的一百万等同于现今的四十万，更别提这是二十多年前！<br/>　　<br/>　　    在时代浪潮的风口是重要的，更重要的还是能找对伙伴。<br/>　　<br/>　　    程燃当即给了蒋舟五万块钱分红，蒋舟拿在手里，有一种烫手山芋的懵逼感。<br/>　　<br/>　　    这小子上了专职就打零工到处做兼职，读书三年下来，挣下了自己的学费，给困难的家庭分担了压力，但手里头何曾一口气拿到过这么多钱。<br/>　　<br/>　　    全程观察了蒋舟的表情，程燃知道自己用人不疑，没有看错。<br/>　　<br/>　　    每天天行道馆的账和现金都是蒋舟收起来的，在他手里过手的钱一个月也是几十万。而他在拿到这五万块的时候表现出了欣喜和振奋，说明了他拥有忠诚和本分。没有因为每天过手那么多钱，而认为程燃赚得多自己拿得少。<br/>　　<br/>　　    什么是该自己的，什么是不该自己所得想都不要想的。<br/>　　<br/>　　    蒋舟心底泾渭分明划着一条线。<br/>　　<br/>　　    和蒋舟结算过后程燃又让二楼的谢飞白五人组到办公室来。<br/>　　<br/>　　    程燃直接拿了大扎现金，拍在了谢飞白等人面前的桌子上。<br/>　　<br/>　　    “根据先前我们的协议，你们有二层楼百分之五的股份，但拥有百分之十的收益权，根据财报，这回收益分红就在这里。”程燃指了指面前的钱。<br/>　　<br/>　　    “那就是我得四万了，你们一人拿两万吧！”谢飞白收了四叠钱塞身上牛仔外套和裤子每一个能装进钱的包里。<br/>　　<br/>　　    其他人纷纷上来拿钱。<br/>　　<br/>　　    他们先前跟着谢飞白盲目投了钱在蓉外门口搞了个咖啡店，结果给亏了个一干二净，现在终于尝到赚钱的喜悦了。<br/>　　<br/>　　    这些个家伙不是富二代就是家里有官职的，这些钱的数额并没有普通学生看来那么遥不可及，所以表现得还好。毕竟他们也曾都拿出压岁钱凑了个十几万和谢飞白去折腾过的。<br/>　　<br/>　　    程燃这个当口问了一句，“拿了钱准备做什么？”<br/>　　<br/>　　    结果一群人早掩不住兴奋的说开了。<br/>　　<br/>　　    “以前都是我爸妈给我钱，现在自己也算是赚钱了，我打算给我爸买件雅戈尔的夹克衫……”<br/>　　<br/>　　    “给我妈买化妆包……去蓉城百货买几支进口口红香水！”<br/>　　<br/>　　    “我爸爱酒，就弄几瓶五粮液茅台……”<br/>　　<br/>　　    看到众人溢于言表，像是秋收收获沉甸甸果实的兴奋……果然还是自己亲手挣的钱，握到的东西，比什么都踏实。<br/>　　<br/>　　    ……<br/>　　<br/>　　    从天行道馆出来，程燃给李明石他们打了个电话。之前李明石说一个他以前的同学，也是高手，奉他感召下前来CQ团队。<br/>　　<br/>　　    CQ是目前程燃种下最重要的一颗种子。<br/>　　<br/>　　    这里初创团队的每一个人，包括了李明石，可能都不知道自己正在参与进行的是一件什么样的事。<br/>　　<br/>　　    或许在未来的某一天回过头看这条长河，他们才会明白当初所做事物的意义。<br/>　　<br/>　　    所以此时加入的每一个人，程燃都要会会对方。8)
