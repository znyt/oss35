　　第440章
<br/>　　当初在程大雷大闹武科场之后，长安上层一团乱麻，许多事先计划好的事情被打乱，被打乱的事情要重新安排。
<br/>　　柳轻名就是被重新安排的人之一。他本来应该在武科场夺魁，到西北军任职，历练几年捞足资本，重新回到长安，自然有一条平步青云的道路等着他。
<br/>　　可这些被程大雷破坏掉了。当然，只因为这一次事情，他的仕途自然不会被打断。后来也进入了军中，只不过，再没有像上次一样的机会。
<br/>　　柳轻名是个很骄傲的人，虽然家里给他铺好了路，但那些路，他并不是很在乎。他只是讨厌失败，在过去的三年里，他卧薪尝胆，勤学苦练，就是有朝一日能洗刷身上的耻辱。假若能重新碰到程大雷，堂堂正正的击败他，那就更好了。
<br/>　　今天终于是碰见了，这是柳轻名的幸运。不幸的是，二人之间的差距并没有因过去的三年时间而缩小，似乎变得更大了。
<br/>　　只用了一招，自己就被他打下马，兵器被夺，铁枪压着肩膀。
<br/>　　对于一个武将来说，这种耻辱可以直接撞墙自杀了。
<br/>　　柳轻名眼中冒火，但此刻却动弹不得。
<br/>　　程大雷喝退了柳轻名的手下人，诸人因为柳轻名被制，手中握着兵器不敢上前，只是包围在周围。
<br/>　　“从马上下来，把你们的兵器都丢在地上。”徐神机走过来，挥舞着短刀喝道：“那个不长眼的，想做小动作，就是自己找死。”
<br/>　　一群人抛下兵器，砸在石桥上砰砰作响。徐神机将他们的兵器都抛进了河水里。
<br/>　　“好啦，现在你们把盔甲脱了，自己也跳进去吧。”徐神机。
<br/>　　“脱衣服？”士兵一楞。
<br/>　　“怎么，难道你们想穿着衣服跳下去，不想再爬上了么？”徐神机道：“银眸，帮帮他们。”
<br/>　　银眸眼睛一亮，忽然抓住一名大汉的肩膀，这人高马大的大汉竟被她举到空中，直接丢进河水中。
<br/>　　哗啦一声响，冬天已快结束，冰水初融，但正是河水最冷的时候。那大汉掉进水中，爆发出一阵杀猪般的尖叫。
<br/>　　“还有谁需要帮忙的么？”徐神机冷声喝道，一副小人得志的猖狂表情。
<br/>　　诸人没有想到，这个小姑娘竟然有如此大的力气。刚开始各人心中都没有将她当回事，此时心中却一阵阵的恐惧。
<br/>　　他们手忙脚乱脱掉身上的盔甲，忍着严寒噗通噗通跳入冰河中。
<br/>　　严寒刺骨，冰水浸润着他们的身体，河中间发出一声声尖叫。
<br/>　　柳轻名当真是一头撞死的心情都有，自己技不如人也就罢了，手下带出来的兵也是如此没用。面对强敌，竟然都顺从的像小羊羔，一个敢反抗的都没有。
<br/>　　这时候，程大雷把压在柳轻名肩膀上的铁枪移开了。但柳轻名却不知道自己该站起来还是继续保持瘫倒的狼狈姿势。
<br/>　　程大雷这是完全没有将自己当会事啊，觉得就算放开自己，自己也威胁不到对方。
<br/>　　柳轻名羞愧得无地自容，然而……事实好像真的如此。
<br/>　　“柳将……一向可好？”最后竟然是程大雷把柳轻名从地上拉了起来。
<br/>　　柳轻名的表情极其尴尬，愣了半晌吐出一句话：“你怎么来了？”
<br/>　　“我过来随便看看，只是没想到在这里碰见你，这实在是太令人愉悦了。”程大雷。
<br/>　　对于柳轻名来说，他可是半点愉悦的心情都没有。这几年时间，不管是相府还是将军府，都一直在收集关于程大雷的情报。柳轻名也知道，程大雷自从逃出长安城后，跑到凉州，又拉起了一支人马。
<br/>　　但他还是完全想不到，程大雷来到并州的目的。
<br/>　　“还有一个问题……”徐神机凑过来：“有吃的么？”
<br/>　　柳轻名对徐神机也有一定了解，知道他是程大雷手下的二号人物，本以为他会问什么重要情报，但他口中抛出来的问题，实在令柳轻名愣了一会。
<br/>　　“这个……有。”
<br/>　　柳轻名从马背上的口袋中取出干粮，三人都饿了一路，坐在一起分食。
<br/>　　柳轻名立在一旁，坐也不是，站也不是。
<br/>　　“要不……你来点儿？”程大雷拖着一张大饼问他。
<br/>　　柳轻名摇摇头，谢绝了程大雷的好意。
<br/>　　程大雷笑笑，道：“现在什么情况？”
<br/>　　柳轻名心中一惊：“你问的是？”
<br/>　　“起义军呐，难道是问你结婚没有？”
<br/>　　柳轻名虽然万般不情愿，但最后还是一五一十对程大雷说了，反正，现在起义军的去向也算不上什么秘密。
<br/>　　林少羽如今的情况十分糟糕，想要从边关突围进入草原，然后又被堵了回来，损兵折将严重。在各路诸侯的围追堵截下，封锁圈越来越小。
<br/>　　“现在他们在哪里？”
<br/>　　“杀虎口。”
<br/>　　程大雷拍拍徐神机的肩膀，道：“好了，咱们该走了。”
<br/>　　三人同时起身，将干粮装在马背上。
<br/>　　“杀虎口的位置？”程大雷问柳轻名。
<br/>　　“向北，五百里。”柳轻名。
<br/>　　程大雷向北方眺望一眼，向北五百里，已经接近幽州，距离青牛山也不远了。
<br/>　　柳轻名愣愣的看着三人，见三人已经上了坐骑，程大雷的坐骑最是恐怖，是一头表情狰狞的黑牛。
<br/>　　“你们去杀虎口要做什么？”
<br/>　　柳轻名现在还不知道程大雷的用意，此时的杀虎口周围已布上重兵，大家要杀林少羽，对于程大雷也不会放过。程大雷这么过去，肯定是羊入虎口。
<br/>　　程大雷没有回答他的问题，三人骑着坐骑，扬长而去，柳轻名耳边，只有咚咚的马蹄声回响。
<br/>　　等程大雷走了，他的手下才哆里哆嗦从河里爬出来。他们浑身发抖，冻得脸色苍白。还有两个人被拽出来后，已经失去了呼吸。
<br/>　　柳轻名看着这一幕，表情极其难看。突然，他一咬牙，提起铁枪，翻身上了马背，纵马向程大雷离开的方向追去。
<br/>　　“柳将，柳将，你要去哪里？”手下人大声叫道。
<br/>　　“杀虎口！”柳轻名没有回头，只是在马背上大喊：“把消息传出去，就说……”
<br/>　　『程大雷……来了』
<br/>(本章完)
