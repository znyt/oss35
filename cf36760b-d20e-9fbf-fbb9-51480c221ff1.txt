　　听到冯经理的话，慕清霄觉得有些搞笑，自己谦让点，有些人还真是把自己当回事，既然不讲理，那还有什么好说的呢？
<br/>　　“我为什么要给你们面子，以为自己是什么东西，看来你们也是蛇鼠一窝，连买块表都靠关系，连经理都这样，这家店也就如此而已。”
<br/>　　话语刚落，冯经理瞳孔一缩，眼中闪过一丝慌乱，这种话要是传到董事长耳中，他绝对是吃不了兜着走，当下开口呵斥道：“小子，你胡说些什么！”
<br/>　　“怎么，敢做不敢让人说，这家店的董事长是不是眼睛瞎，居然让你坐上经理的位置。”
<br/>　　望着周围客户的指指点点，冯经理神色愤怒，在也忍不住，拿出手机道：“保安人员，我这边有两名闹事的，快点过来处理下。”
<br/>　　很快，店外就传来一阵急促的脚步声，只见四名保安人员，手握甩棍，全副武装，风风火火的挤开围观群众，径直朝着店内走来。
<br/>　　冯经理冷笑一声，指着慕清霄道：“你们快点，把这几个闹事的人赶走。”
<br/>　　几秒后，四名保安依旧站在一旁不动，只顾着管理现场，完全不理会冯经理愤怒的神色。
<br/>　　“你们都聋了吗，是不是都不想干了？”
<br/>　　就在这时，门口处走进一名身穿正式西装的男子，男子个子高挑，戴着副眼睛，看起来极为稳重，但眼中且满是阴霾，脸色非常的难看。
<br/>　　男子进入店内，径直走向慕清霄，神色立刻变得极为恭敬，道：“慕老板，我是这家店的总经理，没想到您居然会来我们店里，招待不周的地方，还请您见谅。”
<br/>　　“你们店里服务员的态度和素质都不错，只是一颗老鼠屎，坏了一窝粥。”
<br/>　　慕清霄嘴角勾起一丝不可察觉的微笑，面前的男子他并不认识，至于后者为何认识自己，他也能大致猜到原因。
<br/>　　闻言，男子的脸色更加的难看，愤怒的目光直指冯兆斌：“谁给你的胆子，敢这样对待尊贵的顾客，慕老板，事情我已经听说，保证会给您一个满意的答复。”
<br/>　　冯兆斌见到男子的一瞬间，心中就升起不好的预感，因为男子正是他的竞争对手，这家店的总经理，他虽然也是经理，但后者在任何事情上，都压他一头。
<br/>　　此时，冯兆斌眼中满是骇然之色，能让总经理毕恭毕敬之人，身份肯定不一般，恐怕自己真是踢到铁板了！
<br/>　　“黄......黄经理...误会，这是一场误会！”
<br/>　　冯兆斌想死的心情都有了，他只是想和贾老板联络下感情，没想到却闯了大祸，不但得罪了大佬，这份工作恐怕也保不住了。
<br/>　　总经理语气冷冽道：“冯兆斌，从今天开始你就不用在这里工作了，立刻收拾收拾，马上去财务部结算这个月的工资，我们店里可容不下你。”
<br/>　　“别......总经理，我知道错了，我真的知道错了，是我有眼无珠，是我瞎了狗眼，请你在给我一次机会，我真的不能丢了这份工作啊！”
<br/>　　冯兆斌脸色惨白，快速跑到吴耀鹏面前，跪下抱住他的大腿，一脸哀求的说道。
<br/>　　吴耀鹏直接将他一脚踹开，冷声道：“你不能丢了这份工作，我还不能丢了这份饭碗呢！”
<br/>　　他可不是傻子，上次他跟董事长参加静海市晚宴，可是亲眼看到，面前这位男子可是欧阳紫嫣的男人！
<br/>　　欧阳紫嫣是谁，那可是首都欧阳家的公主，他们董事长都高攀不起的人物，能当她的男人，身份可见一斑。
<br/>　　脸色惨白的瘫软在地上，冯兆斌瞬间将视线落在慕清霄身上，旋即又看了看王语嫣两人，神色颓废的低下头去，他知道自己这辈子都完蛋了。
<br/>　　冯兆斌的事情应验了一句话，可怜之人，自有可恨之处。
<br/>　　很快，冯兆斌就失魂落魄的离开名表店，再也不是什么经理。
<br/>　　待后者离开后，吴耀鹏看向女服务员笑道：“小刘你做的很不错，从今天开始你就去财务部把，财务部总监这个位置很适合你。”
<br/>　　闻言，女服务员眼中满是不可置信，真是没想到，自己不但没有被开除，还成为财务部总监，简直跟做梦一样。
<br/>　　“谢谢总经理，谢谢慕老板。”
<br/>　　等女服务员离开后，吴耀鹏方才献媚的看向慕清霄：“慕老板，这样处理，您觉得还满意吗？”
<br/>　　听到吴耀鹏的话语，慕清霄指着中年男子和他的小情人，似笑非笑的说道：“那这两人怎么处理？”
<br/>　　吴耀鹏的目光落在两人身上，笑道：“哟，这不是贾老板吗，今天什么风把你给吹来了？”
<br/>　　此时，贾权额头已经出现细汗，心中有些慌乱。
<br/>　　他在这个圈子里混的时间也不短了，地位虽然不是很显赫，但勉强算的上有钱人。
<br/>　　可是吴耀鹏的话语和态度，他刚刚都听的清清楚楚，二话不说就将冯兆斌给开除了。
<br/>　　若面前的青年只是个普通人，打死他都不信。
<br/>　　有钱确实可以满足自己很多的欲望，但钱也不是万能的，在绝对的权利面前，金钱最多也就是一堆废纸罢了。
<br/>　　贾权抹了把额头的冷汗，脸上露出一个比哭还难看的笑容：“慕......慕老板，刚刚是我有眼不识泰山，你就当我是个屁，给放了吧，这只表算我孝敬您的。”
<br/>　　“怎么，你的女人不是想要这只表吗？”慕清霄嘴角勾起一丝戏谑的笑容，饶有兴致的望着贾权。
<br/>　　原本，他认为女人变脸比翻书还快，现在却发现自己忽略了一点，有时候男人也可以如此。
<br/>　　面前这位肥头大耳的暴发户，就是典型的例子。
<br/>　　听闻慕清霄的话，贾权脸色更加白上几分，身上的肥肉都跟着颤抖起来。
<br/>　　而他的小情人则躲在身后，低着脑袋丝毫不敢抬头，心中非常后怕，自己的并头都这种态度，面前的男人不会是某位公子哥把！
