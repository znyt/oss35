        “嗯？想杀我？”傅夜沉黑眸微眯，大手像钳子一样，一把掐住了童筱颖的咽喉，“说，是谁指使你来的？夏阳枭？还是连中凯（连华生的父亲）？或者是那个冒先生？”
<br/>
<br/>        “……”童筱颖只觉一口气堵在喉咙里出不来，她惊恐地使劲摇头，以求换取一丝生机。
<br/>
<br/>        僵持不下时，她鼻子一酸，眼泪瞬间在眼眶里打转，心口上开始一阵又一阵地发痛。
<br/>
<br/>        傅夜沉微蹙的剑眉越锁越紧，黑眸凝视着她眼角滑落的泪珠，不禁心软了下来。
<br/>
<br/>        不管她到底是第几人格，她始终都是筱颖，他深爱的，却不知道如何去爱的，他失而复得的，却不知道该怎么去珍惜的筱颖。
<br/>
<br/>        手下的力度渐渐地松懈，傅夜沉放开了童筱颖，坐直了身子，将裤袢上的手枪抽了出来，递给了这个女人。
<br/>
<br/>        “你不是想杀我吗？好！我给你这个机会！来，杀了我。”傅夜沉拿着枪口，将枪柄指向童筱颖，示意她握住手枪向自己开枪。
<br/>
<br/>        童筱颖呛了几声，双手支撑着身子，坐起来，并向后挪了挪，直到后背靠到了床头板上无路可退，才稳住身子，支支吾吾地回答道：“对不起，我、我没想过要杀你的。我只是……”
<br/>
<br/>        只是什么呢？
<br/>
<br/>        她说，她只是被潜意识支配了，这种话，他会信吗？
<br/>
<br/>        从她得知他的名字那一刻起，她脑海里就有很多意识不受自己的控制。
<br/>
<br/>        “既然你不杀我，那么轮到我来杀你了！”傅夜沉的声音冷酷无情。
<br/>
<br/>        银色的手枪在他修长的指间转了半圈，他娴熟地握住枪柄，将枪口抵在了她的额头上。
<br/>
<br/>        童筱颖吓得脸色惨白，难以置信地瞪大了眼睛。
<br/>
<br/>        “嘭”地一声，枪声响起。
<br/>
<br/>        童筱颖闭上了眼睛，吓得双手抱头，身子蜷缩成一团，浑身颤抖不停。
<br/>
<br/>        子弹只是打在了她身后的墙壁上。
<br/>
<br/>        傅夜沉从容不迫地收起手枪，冷斥道：“给我滚，不要再让我见到你。”
<br/>
<br/>        童筱颖从惊魂未定中清醒过来，睁开眼睛，抬起头来时，她只觉面前这个英俊的男人突然间变得好陌生，甚至令人不寒而栗。
<br/>
<br/>        “我、我离开你，不、不知道自己要去哪里。”
<br/>
<br/>        沉默了半晌，她十分委屈地诉说。
<br/>
<br/>        难道要她继续回到那个实验室里，被那群穿着白马褂的医生每天抽血拿去做研究吗？
<br/>
<br/>        至少，她待在他身边，可以像个正常的人一样活着。
<br/>
<br/>        “呜呜——”一想到这里，童筱颖既心酸又委屈地抽泣起来，“我都说了我没想过要杀你，你都还不信，还要赶我走！你是我哪门子的大哥啊？只知道凶我！”
<br/>
<br/>        “……”傅夜沉顿时黑脸了。
<br/>
<br/>        “我告诉你，我就不走，打死我也不走！你让我滚，我就滚啊！那我多没面子啊！我才不滚！我就赖着你，哼！呜呜——”童筱颖忍不住揉着眼睛哭诉起来。
<br/>
<br/>        傅夜沉曲起长腿，手肘抵着膝盖，有些头疼地抚了抚额头，然后一脸无奈地看着面前这个一边哭泣一边撒泼的小女人。
<br/>
<br/>        明明是一件很严肃的事情，结果变成了现在这个样子。
<br/>
<br/>        不管怎么说，她也是自己的女人。
<br/>
<br/>        总不能任由她一个人哭花了脸。
<br/>
<br/>        傅夜沉收敛了刚刚的冷酷，语态稍稍放温和了些，说道：“行了，别哭了，你想留下，就留下来好了。”
<br/>
<br/>        “你还这么凶！”童筱颖决定还是继续哭一会儿好了，这个男人太凶了。
<br/>
<br/>        更何况，刚刚他还把她给吓到了。
<br/>
<br/>        傅夜沉叹了口气，伸手抓住童筱颖的手腕，将她往自己怀中一拉。
<br/>
<br/>        她整个身子，硬生生地撞入他的怀中。
<br/>
<br/>        童筱颖本能地仰头，不约而同地迎上了他俯下身来的吻。
<br/>
<br/>        哭声戛然而止。
<br/>
<br/>        她的脑海里，出现了似曾相识的画面。
<br/>
<br/>        曾几何时，她哭得伤心的时候，他也是这样一边吻着她，一边哄着她的。
<br/>
<br/>        在童筱颖惊愕得瞪大的瞳仁里，这张微凉的薄唇摩挲着她的粉唇。
<br/>
<br/>        他长臂紧紧地揽过她的肩头，将她柔软的身体像抱着毛绒娃娃一样扣在自己怀中，并低头轻柔地啃咬着她的唇瓣，偶尔他又将舌头伸进她微启的口中，霸道地席卷她所有的甜蜜。
<br/>
<br/>        “乖，终于不哭了。”傅夜沉温柔地呢喃道。
<br/>
<br/>        鼻端涌入的来自她身上的气息，就像是一种致命的催情毒药，让他的腹下，对她有了一股燥热难耐。
<br/>
<br/>        筱颖就是筱颖，不管她变成什么样子，他的身体总会对她表现出最忠诚的反应。
<br/>
<br/>        童筱颖缓缓地闭上了眼睛，纤纤的小手，慢慢地爬上这个男人的衣襟，不急不缓地替他解开衬衣上一粒又一粒的纽扣。
