　　所有的一切都发生在刹那间，在场的众人都震惊当下，不少暗杀者便就此趁机冲入，延着那两名先一步破开防守圈的地带直奔，同时一阵阵刀光剑影，到处都是打斗声。
<br/>　　防守队这么辛苦坚持下来的重重防御圈就这么被打破，中间被强势破入的那一片地带死伤最为惨烈，地上全是被染红的土地，渗入地底带起泥泞感。
<br/>　　那些暗杀者可不会放过如此大好机会，最幸运的一个甚至已经来到木屋的窗边，就想要破窗而入！
<br/>　　顾凉晨和尚九娣同时返回木屋，用身躯将一切可以破入的地方都堵死，无论是门窗还是细缝。
<br/>　　那些暗杀者还穿着可以变幻的外套，视觉在无用的情况下，两人的听觉和嗅觉感知提升到了极致，或许是在绝对的信念之下总会有奇迹发生，竟然一抓一个准，硬生生的将两名隐身潜行的暗杀者揪出来！
<br/>　　这些都发生的太快，让很多狼牙成员始料未及，战斗还在继续，双方都在死扛。
<br/>　　而在木屋门前，白允儿左肩带着那半柄长剑，无视了那触目惊心的腥红鲜血流淌，至始至终她别说喊疼，连眉头都没有皱一下。
<br/>　　眼前的暗杀者还没调整好心态，从一开始他就没想过能一击命中白允儿，毕竟在这次任务里白允儿属于终极目标一样的存在，下达任务之时也被交代过，白允儿能杀便杀，不能杀让她吃点苦头便作罢，先行缠住后让其他人能去杀楚涵。
<br/>　　所以在命中白允儿之后，这名暗杀者完全愣住，有些不可置信又内心狂喜，而与他一起行动的另一名暗杀者，也在旁边忘了接下来该干什么，彻底被眼前的一幕所惊。
<br/>　　有谁见过白允儿流血受伤？
<br/>　　连白忧都抓不住也没办法处理的人，竟然就这么轻而易举的被刺穿左肩！
<br/>　　于是在这样的震惊之余，两人便就此错失破门而入的最佳时机。
<br/>　　在四周都爆发了激烈死斗之际，白允儿猛然身躯朝前右手伸出，就这么直接抓向眼前这名暗杀者的喉咙，这是她惯用的攻击手法，相当的暴力血腥，破喉而入然后拧断脑袋！
<br/>　　那名暗杀者连忙反应过来，步步后退的同时，手中残剑抬起格挡。
<br/>　　这可不是普通的武器，来自于神秘家族培养的死士，手中剑刃都异常锋利，当然若非白允儿并不是纯血脉的白家成员，没有受到星河法则的庇护，再锋利的武器也刺不中她，只可惜血脉的不纯还是导致她因此受伤。
<br/>　　这一点已经不是秘密，眼前的这名暗杀者自然清楚，所以在格挡之时也一瞬间的表情狰狞，白允儿徒手抓来一定会被这柄残剑的剑刃所伤，而他到时候再反攻用力，断掉白允儿的手并不成问题！
<br/>　　就在这名暗杀者如此设想，也做好了反攻准备之时，白允儿的手已经来到近前，只是并不像他预料的那样发展，眼前这只手并非常见的素白青嫩，而是闪着一丝古怪的银光。
<br/>　　怎么回事？
<br/>　　他皱起眉，但在这电光火石之间根本来不及细看，只听得‘叮’的一声脆响，紧接着——
<br/>　　嘭！
<br/>　　他格挡用的残剑猛然再次断成两截，一部分掉落在地发出‘哐当’的声音，一部分则是短到了剑柄处被他握在手中。
<br/>　　而白允儿的右手，则已经在刹那间‘噗嗤’一声破入他的咽喉！
<br/>　　暗杀者的面上还带着震惊和恐慌之色，他甚至来不及去思考这一切是怎么回事，就瞬间被白允儿结束生命，飙飞的鲜血喷洒的到处都是，星星点点的将干净的木屋门都弄脏。
<br/>　　白允儿一把抽·出深入此人咽喉的手，带出一道猩红色的黏稠液体染上她的白色裙摆，紧接着左腿顺势轻轻一踏。
<br/>　　嘭！
<br/>　　这名暗杀者就这么倒在她的脚边，头颅呈现不正常的偏离状态倒向一边，身躯毫无力度的瘫在地上一动不动。
<br/>　　从白允儿出手到击杀，再到将这尸体扔沙袋一样的扔在一边，从头至尾她的眼神甚至都没往地上看一眼，而是结束之后立即两步上前，朝着旁边另一名还属于神隐状态的暗杀者攻去！
<br/>　　那名暗杀者大惊，白允儿的出手太过于快速，同伴的死亡更是在眨眼之间，让他没有任何机会反应。
<br/>　　此时当他还处于隐身之中，白允儿却好似能看到他一样，直奔着他的面门而来，那只带有其同伴鲜血的右手，也就这么直接抓向他喉咙时，这名暗杀者只觉得浑身寒如冰窖，脑袋好似被什么东西死死禁锢，动弹不得！
<br/>　　噗！
<br/>　　又是一声破入血肉的声音，连带着这名隐身的暗杀者身形顿时显露而出，当周围人能看到这名暗杀者的时候，便是与白允儿一只手深入其喉咙的场景一起。
<br/>　　何等的暴力直接！
<br/>　　快速又狠厉的战斗让不少狼牙成员大开眼界，更惊呆了不少前来的暗杀者，两批来自不同家族的队伍纷纷下达了撤离木屋的指令，开始采取了迂回模式打算拖延时间。
<br/>　　所有人都很清楚，白允儿很强，强的根本不能与之一战！
<br/>　　但同时……
<br/>　　她也受伤了，并且受伤不轻！
<br/>　　为什么两次出手都是右手，因为她的左肩处还插着一柄没入血肉的剑尖！
<br/>　　她的每一次行动，每一次出手，每一个动作甚至表情，都会牵扯到左肩处的伤口，那可不是小伤，那柄剑尖距离她的心脏不远！
<br/>　　所以两批暗杀者同时采取的一个决定，便是拖！
<br/>　　拖到白允儿无法继续战斗为止！
<br/>　　只要白允儿无力一战，无论是流血过多还是战力大减，剩下的这些狼牙防守队成员都不足为惧，他们承认狼牙很强，哪怕是这些没能选上与狼牙战团一起上战场的最后一批防守成员，也强大的让他们死伤难看。
<br/>　　但只要楚涵一死，这一切的牺牲就都值得！
<br/>　　这些人的强大，狼牙战团的强大，都是拜楚涵所赐，楚涵不在，从此便没有可以再让他们受挫的队伍！
<br/>　　在外面爆发着这些战斗的时候，木屋内的却是极其安静，步纱已经在床边紧闭着眼睛坚持了好几天，而躺在床·上的楚涵更是如同死人一样一动不动。
<br/>　　只是在没有人知道的黑色封闭空间里，楚涵却是对外面发生的一切都听的一清二楚，尤其是距离木门最近的那场战斗。
