　　“同时炼制十炉丹药，这得需要多强的精神念力？”
<br/>　　“而且还是十种不同的丹药，控制力同样精准的可怕！”
<br/>　　“不过，就是不知道，这位叶公子炼的几品丹药？二公子那边的那位柳大师，据说每天可以炼制十颗四品中等丹药，三十多颗四品下等丹药呢！”

<br/>　　就算是一千颗一品丹药，也比不上一颗三品丹药，数量再多，炼丹手法再炫，也不过是哗众取宠罢了！
<br/>　　很快，
<br/>　　十尊丹炉里的丹药，几乎同时凝聚成形。
<br/>　　咚咚咚！
<br/>　　丹炉落地，在地上排成了一排。
<br/>　　张凌云连忙带着他手下的炼丹师，快步冲了上来，

<br/>　　“乾元四气神丹，也是四品上等！”
<br/>　　“太阴凝碧丹，四品上等！”
<br/>　　“度厄金丹，四品上等！”
<br/>　　……
<br/>　　十炉丹药取出，竟然全都是四品上等，而且足足一百多颗！
<br/>　　众人顿时再次目瞪口呆。
<br/>　　要知道，纵然是四品炼丹师，所能炼制的四阶灵丹种类，一般也不过就是比较拿手的几种罢了。
<br/>　　叶尘同时炼制十炉丹药，而且全都四品上等，这简直令人难以置信。
<br/>　　难道他真的是五品，甚至更高级别的炼丹师？
<br/>　　“哈哈哈哈！”
<br/>　　张凌云愣了片刻之后，顿时兴奋的哈哈大笑起来，
<br/>　　“叶兄弟，我果然没有看错你！有了这些四品上等丹药，我看老二他还怎么和我争？那个什么狗屁柳大师，和叶兄弟你一比，简直就是一个废柴啊！哈哈！”
<br/>　　叶尘却一脸平淡，
<br/>　　“我要的治愈神魂类的灵草，为何还没有送来？”
<br/>　　张凌云顿时露出一抹尴尬之色，连忙赔笑道：
<br/>　　“是我办事疏忽了，我这就让人立即送过来！”
<br/>　　……
<br/>　　与此同时，
<br/>　　张齐云所在的院落之中，
<br/>　　“这次多亏有柳大师相助，拍卖会的生意才会如此火爆，等我打败了老四，成为了万宝阁的少阁主，一定重重相谢！”
<br/>　　坐在首位上，是一名须发花白、仙风道骨的老者，应该就是那位四品炼丹师柳天生了。
<br/>　　柳天生捋了捋花白的呼吸，笑呵呵道：
<br/>　　“二公子不必客气！老夫既然是您的客卿，自然就要为您尽一份力！”
<br/>　　说着，
<br/>　　柳天生忽然长袖一挥，
<br/>　　唰！
<br/>　　一个精美的青色丹瓶，出现在桌子之上。
<br/>　　张齐云见状，顿时两眼一亮，喜道：
<br/>　　“柳大师，莫非您老人家，又炼制出什么新的灵丹妙药了不成？”
<br/>　　柳天生满脸傲然之色，缓缓道：
<br/>　　“二公子，此丹名为度厄金丹，筑基境巅峰境的修士服用，可以提升突破金丹的概率，乃是四品上等灵丹！”
<br/>　　张齐云闻言，顿时一拍桌子，满脸狂喜之色，
<br/>　　“恭喜柳大师的炼丹术又进一步，如今连四品上等丹药都能炼制出来，相信要不了多久，应该就可以突破五品了吗？”
<br/>　　柳天生哈哈一笑，摆了摆手道：
<br/>　　“哪有那么容易！这度厄金丹，老夫炼制十次，也不过才能侥幸能成功一两次罢了！离五品炼丹师，还差得远呢！”
<br/>　　柳天生嘴上这么说着，眉眼之间，却全都是得意之色。

<br/>　　张齐云连忙道：
<br/>　　“柳大师过谦了！谁人不知道，您老人家出自丹域，不满百岁就已经是四品炼丹师，即便是在丹域之中，也是了不得的人物，将来晋升五品，乃至六品，根本不在话下！”
<br/>　　柳天生哈哈一笑，
<br/>　　“如此，那我就借二公子的吉言了！”
<br/>　　……
<br/>　　就在两人正相谈甚欢之时，张齐云的一名手下，忽然急匆匆的冲了进来，
<br/>　　“二公子！大事不好了！”
<br/>　　张齐云眉头一皱，冷喝道：
<br/>　　“没看到我正在陪柳大师喝酒吗？慌慌张张成什么样子？”

<br/>　　“二公子！是、是拍卖场那边出事了！我们的顾客，现在全都跑去四公子那边的拍卖场了！”

<br/>　　张齐云听到这话，顿时两眼一瞪，直接霍然站起身来，

<br/>　　“我听人说，四公子那边的拍卖场，刚刚不不久前，推出十种不同的四品上等丹药，足足有上百颗，现在整个玄天城的权贵子弟，都快要抢疯了！”
