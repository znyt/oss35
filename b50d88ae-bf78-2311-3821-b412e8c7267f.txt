
<br/>　　<br/>　　    王惊蛰默默的看着钓鱼姑娘为老赵家招魂，他没想到对方居然也是同道中人，并且手法还挺地道的，绝对不是初出茅庐的生瓜蛋子，之前在渭河上钓鱼的时候她就像个邻家女子一样显得特别安静和祥和，那种恬然的气质莫名的就让自己脸红了。<br/>　　<br/>　　    王惊蛰脸红首先是因为冲着人家撒了一泡尿，其次是觉得他好像亵渎了那姑娘，就好像一片白纸摆在面前他却不小心在上面甩了一串污点，太唐突了。<br/>　　<br/>　　    其实王惊蛰不知道的是，有时脸红，也是因为怦然心动。<br/>　　<br/>　　    放在地上的引魂灯忽明忽暗的飘摇着烛火，老赵的儿子穿着他爹的衣裳，冲着河面的方向忽然开口喊道：“赵常柱，赵常柱······赵常柱，你快回来吧”<br/>　　<br/>　　    “呼！”从渭河河面忽然飘过来一阵阴风，水面悄然间就开始涌动起来，随即一阵“哗啦”的水流声渐渐的传了过来。<br/>　　<br/>　　    钓鱼姑娘突然开口说道：“一会你们不论看见什么都不要太惊慌了，没什么事的”<br/>　　<br/>　　    这姑娘的话音刚落，那“哗啦，哗啦”的水流声就更急促了，河水中忽然间就冒出个影子，穿着身蓝色的工服趟着水就走了过来，这人影浑身湿哒哒的，就好像是在水里泡了很久一样。<br/>　　<br/>　　    身上浮肿，脸色苍白，七窍滴答的往下流着血迹。<br/>　　<br/>　　    老赵的媳妇和孩子顿时就被吓了一跳，尽管看出是老赵了，但还是忍不住被吓的往后一连退了好几步，老赵仿佛没有神志一样，懵懵懂懂的就朝着岸边放着的引魂灯走了过来，来到近处后就呆若木鸡的不动了，眼睛直勾勾的看着飘摇的烛火。<br/>　　<br/>　　    老赵刚死没多久，这个时候他的魂魄还是不稳的，灵智还不太健全，影子虚虚实实的好像来阵风就能给吹散了一样。<br/>　　<br/>　　    老赵的媳妇和家人惊恐的看着他，捂着嘴巴差点就要尖叫出来了，钓鱼姑娘伸出一根手指“嘘”了一声：“不要嚷嚷，不然你们会吓到他的，站着别动看着就行了，我要问他几句话”<br/>　　<br/>　　    这女子说完也不等赵家的人回话，就来到老赵的身前，弯腰拿起引魂灯将一张符纸递到了烛火上，瞬间烛火烧的就旺了起来，老赵的魂魄也似乎渐渐的稳了不少。<br/>　　<br/>　　    自然，这烛火要是灭了的话，他的魂也就散了。<br/>　　<br/>　　    “赵常柱当时在河里是怎么一回事·······”<br/>　　<br/>　　    赵常柱听见问话后茫然的张了张嘴，仔细思索了半天，脸上的神情顿时就惊恐起来，五官都狰狞在了一起：“有，有人拉我下去，他，他死死的拽住了我的脚，我，我怎么都挣扎不开，然，然后就被他给拉进了水里，就再也出不来了·····有人拽我，一直在拽着我，我拼命的往下游也游不动”<br/>　　<br/>　　    老赵的而在旁边顿时叫道：“不可能，当时的水浪那么大，河底下怎么可能会有人的？”<br/>　　<br/>　　    钓鱼姑娘瞪了他儿子一眼，对方被吓了一跳抿着嘴不吭声了，她又接着问道：“你的尸体在哪呢，飘走了，还是在河下？”<br/>　　<br/>　　    老赵的身影往后看了一眼，抬手又指了指，钓鱼姑娘“哦”了一声，示意自己懂了，随即老赵魂魄渐散。<br/>　　<br/>　　    从老赵出现到消逝前后也不过就几分钟的时间，他的魂魄挺不了多久，这是他死的第一天，等到老赵头七的时候他的魂恐怕要比现在坚韧得多了，甚至完全有可能蜕变成冤魂，因为老赵的死确实有点冤了。<br/>　　<br/>　　    “你们先回去，明早过来收尸······”钓鱼姑娘想了想，又有点不确定的说道：“差不多能收到吧”<br/>　　<br/>　　    老赵媳妇诧异的问道：“不用守灵了么？”<br/>　　<br/>　　    “魂都已经来过了，还守的什么”钓鱼姑娘皱眉说道。<br/>　　<br/>　　    随后，老赵的媳妇和家人收拾了下就走了，渭河岸边这个时候也没什么人了，这女子来到河岸前，轻车熟路的解开一叶木舟。<br/>　　<br/>　　    王惊蛰就在旁边看着，从头到尾也没插话，见她好像要上船划向河中就忍不住的问了一句：“你是打算去捞老赵的尸体？”<br/>　　<br/>　　    “也是，也不是，我想看看这渭河下到底有什么东西”钓鱼姑娘解释道：“我小的时候就生活在这里，长大以后就喜欢在河上划船钓鱼，我算是被这条渭河养育大的，所以我真不想看到这河里有什么东西，扰乱了两岸人的生活，捞上老赵的尸体我得看看是什么东西把一个生活在渭河边上几十年的老船工都能给淹死了”<br/>　　<br/>　　    “那你总归也得找个捞尸工下去吧？”<br/>　　<br/>　　    “他们敢么？”姑娘反问了一句。<br/>　　<br/>　　    王惊蛰顿时语塞。<br/>　　<br/>　　    姑娘解开船绳后，就把木舟推进了河里然后上了船，王惊蛰憋了半天最后终于忍不住的说道：“我跟你一起过去吧，你一个人······有点危险”<br/>　　<br/>　　    “上来吧”姑娘也没拒绝，想必知道王惊蛰从头到尾都留在这没走，肯定也是会点什么的。<br/>　　<br/>　　    “哗啦”木舟入水，女子熟练的划着船桨说道：“老赵出事的那一片水域不深，也就五六米左右，能见度肯定是看不见的了，不过我差不多能根据他刚刚的魂魄感应到尸体在哪一带”<br/>　　<br/>　　    王惊蛰哦了一声也没接腔，女子瞥了他一眼也不再吭声了，寂静的渭河上一条小船载着两人没用多久，就到了之前翻船的地方。<br/>　　<br/>　　    这女子停下船桨，挽起袖子后就把头发给扎上了，王惊蛰看着她白嫩的手臂忽然之间就愣了愣。<br/>　　<br/>　　    钓鱼姑娘右手的小臂上有一颗鲜艳的红痣十分显眼，大概比小拇指盖小一点。<br/>　　<br/>　　    王惊蛰知道那不是痣，叫守宫砂，古时未经人事的女子或者要进宫当妃子的女人都会在小臂上点上一颗守宫砂来表明自己的贞洁。<br/>　　<br/>　　    只不过，这个习惯大概最近百年都已经不存在了，现在更是根本就见不到了，王惊蛰好奇的是这女人的身上怎么会有此习惯。<br/>　　<br/>　　    钓鱼姑娘扎好头发后就站了起来，看着水面准备要跳下去的时候，忽然转身说道：“按照正常情况来讲，这个时候你不是应该说，让我下去吧，更合适一点么？”<br/>　　<br/>　　    王惊蛰：“·······”<br/>　　<br/>　　    看着王惊蛰一脸懵逼的表情，姑娘噗嗤一声笑了：“逗你玩的呢”<br/>　　<br/>　　    王惊蛰的脸又红了，他憋了半天，吭哧着说道：“我不会游泳的”
