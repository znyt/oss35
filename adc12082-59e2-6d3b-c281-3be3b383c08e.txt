　　在游历混沌这些岁月里，元释也不是什么也没找到，那些顶尖的灵宝灵根他并没有遇到，但是混沌之中一些普通的灵宝、材料倒是遇到不少！
<br/>　　混沌是万物之母，孕育着无穷造化，这其中的宝物在混沌之中显得寻常，要是放在后世，每一件都会让无数修士杀得眼红。
<br/>　　能够在混沌之中存在的东宝物，至少也得是先天灵宝级别的。不过，这些对于元释来说，却是没有多大作用，以他如今的实力，至少也得混沌灵宝级别，才对他有帮助。
<br/>　　只是这混沌灵宝以上的宝物，即便是混沌之中也十分稀少，大多数还是混沌魔神们的伴生灵宝，即便元释为前十出世的魔神，也没有。
<br/>　　但就算如此，他还是收了一些，毕竟最后混沌还是被盘古毁了，这些宝物要是跟着毁了，岂不可惜？谁也不会嫌自己宝物多？
<br/>　　有了这些神材、宝物的加入，元释自己所炼的因果莲台也升级到了先天至宝级别，而且还是顶尖的先天至宝，不过，就算如此，这对于元释的实力增长还是有限的。
<br/>　　……
<br/>　　岁月无情，不知过了多久，元释一直在这混沌之中行走，感悟，他的修为并没有突破，倒是他的魔神真身得到了进一步的突破，肉身就堪比混元十重天的强者。
<br/>　　随着他猎杀的混沌魔神越多，元始印也越来越完善。虽然没有更多时间来感悟其他法则，但其中一些法则的简单运用，他还是会的。这使得他的实力更进一步！
<br/>　　“咦！前面怎么有如此浓郁的混沌之气？”元释看见前方混沌之气暴烈翻滚，非比寻常，像是在孕育这什么，应该有什么宝物存在。
<br/>　　几步跨出，元释就接近目标。
<br/>　　越接近目标，混沌之气愈发的浓郁，而且还带着强烈的罡煞攻击，要不是元释的魔神真身强悍，还不一定能坚持下来。
<br/>　　无数罡煞之气、混沌之气全部往此地汇聚，只进不出。元释顶着强烈的罡煞之气，坚持向罡煞中心走去，他特别好奇这混沌之气中心到底是什么东西，怎么会有如此多的罡煞聚集在此。
<br/>　　“好厉害的罡煞！以我如今堪比混元十重天的魔神真身都有些承受不住？”元释感觉刮在身上的罡煞很是疼痛，差点忍受不住。不过，这样越发的激发元释内心强烈的好奇心，这地方不会有什么至宝存在吧？
<br/>　　“越来越厉害了！”越是靠近，也就越是难受，但元释还是坚持下来了，并且他也不急，这里有如此多的罡煞之气，趁此机会磨砺自己的魔神真身。因为他发现刚突破不久的魔神真身在这浓郁的混沌之气及强烈的罡煞之中任能进步，这种机会十分难得。距离混沌劫已经越来越近了，这时候能增加一分实力就多了一分自保能力，实力越强越好。
<br/>　　元释吸收着这里的混沌之气，并将自己的因果莲台取了出来，让它自行吸收这些混沌之气，提升品级。
<br/>　　一步，
<br/>　　两步，
<br/>　　三步，
<br/>　　……
<br/>　　越到中心，罡煞越密集，元释也越难进步，同时身上忍受的痛苦也就越重。当然，魔神真身的进步也就越发的明显了。有时候每前进一步，都要停在原地淬炼好多年，等到自己适应了，就再次往前踏进一步。
<br/>　　轰！
<br/>　　全身上下散发出十道耀眼的青光，这是不灭青光，只有在混元圆满时才会出现。不知过了多少年，元释的修为再进一步，终于突破了困扰已久的混元大罗金仙九重天，达到了混元圆满初期的混元大罗金仙十重天，也就是第三步最后阶段了。对于因果法则的掌控也更近一步，都快要到凝聚因果大道法则本源的地步了。一旦凝聚因果法则本源，那就突破第三步，开始迈进第四步——天道境！
<br/>　　同时，元释的魔神真身更是再进一步，达堪比混元十一重天的强度。
<br/>　　修为的突破，让元释领先别的魔神一步，同时也让他在最后的混沌劫中多了一点保命的实力。修为的突破，面对这些**的罡煞，元释已经不再畏惧了，他一步一步，马上就走到了罡煞的中心，同时，也发现了其中心的宝物。
<br/>　　跨过混沌罡煞旋涡，元释定睛一看，居然是一株灵根，这是一株古茶树，树皮干裂，如龙鳞张开，枝干不过半米粗，形似一条虬龙盘卧，蜿蜒伸展向半空而去，最高处离地不足三米。
<br/>　　满树晶莹，每一片叶子都不一样，流动着道的光辉，它是如此的奇特，小鼎、神凤、仙人、神钟、山河、云朵、神凰、八卦、太极、太阳、月亮、星辰，栩栩如生，每一枚叶子都像是一个小世界，代表着一种与众不同的道，摇曳出梦幻的色彩。
<br/>　　各种叶子，奇形怪状，皆有独特而宝贵的道韵，一树灿烂，分外耀眼，让人迷失。
<br/>　　元释一接近古树，心里就空灵了下来，方才的浮躁、不安全部消失了。内心一片祥和，无喜无忧，忘记了一切烦恼，让人感到平和与安谧。
<br/>　　古茶树树干是古老的，充满大气与古意，而叶子却是自然的，充满了生命与道韵，交融在一起，神秘而奥妙，玄之又玄！
<br/>　　“一，二，三……”这株古树叶子很少，不过百余枚而已，但却并不显得光秃，因为每一片叶子都与众不同，让人忘了其他。
<br/>　　一共有一百零八片叶子，不多不少！
<br/>　　混沌悟道古茶树！
<br/>　　看见这一株混沌灵根，元释心中浮现出一股信息：混顿悟道古茶树！生于混沌，食混沌之气，每一会元（十二万九千六百年）换一次树叶，共计一百零八枚，每一枚树叶代表着一种不同的道，为悟道之树！
<br/>　　“混沌悟道古茶树？不错不错，游历混沌如此长时间，终于找到了一株混沌灵根了，真不容易啊！”元释望着眼前正在吸收混沌之气的悟道古茶树，感慨道。
<br/>　　“不过，这个吸收混沌之气是个麻烦，我身上可是没有那么多的混沌之气来供养它啊！”元释突然又想到一个问题，现如今在这混沌倒好，要是今后开启洪荒，没有了这混沌之气又要怎么办，到时候自己可没地方哭了。
<br/>　　元释想了想，还是没有办法解决，
<br/>　　“呃！不管了，还是趁此机会在此地修炼一翻，正好有这罡煞之气的阻拦，此地又有混沌悟道古茶树的帮助，再加上鸿蒙元始印，我就不相信我不能悟道。”
<br/>　　说完，元释来到古茶树旁，魔神真身快速缩小，变成一个两米左右的男子，一步跨出，盘腿在混沌古茶树下坐下。
<br/>　　摘下一枚古茶树也，含在口中，一种玄之又玄的奥妙弥漫全身，瞬间元释就陷入了悟道之中，不可自拔！
<br/>　　在其周围一朵黑白两色莲台飘在半空，缓缓旋转，吸收着周围蜂拥而来的混沌之气！
