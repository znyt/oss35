　　如果是带机关的门，不管是什么机关，总是有迹可循，但是锁孔的话，没有钥匙，谁也别想打开这扇笨重的石门。
<br/>　　“你祖先，没给你留下钥匙什么的？”叶少阳望着妙心问道。
<br/>　　妙心歪了歪脑袋，看她的表情就知道没有了。
<br/>　　叶少阳伸出手，在石门上下敲了敲，发出略微清脆的声音，由此说道：“这门并不厚，而且石门一般都脆，如果有什么足够坚硬的东西，应该可以撞的开。”
<br/>　　转头在墓室里又看了一圈，实在是什么都没有，一向也是，制造这座石门的人不是傻比，如果还留下东西用来给人撞开石门，那还不如不装这个门了。
<br/>　　这时候毛小方想到一件事，对妙心说道：“外面的石门是开着的，但是这里的却关着，按说没这个道理，只有一种可能，你祖先根本没从这座石门进出。”
<br/>　　妙心道：“话是没错，可是这里根本没有别的入口。”
<br/>　　毛小方转头看向了那口水井。
<br/>　　妙心立刻会意，皱眉道：“不可能吧？”
<br/>　　“除非你能找到别的入口，不然就只有这个水井了。再不可能也是它了。”
<br/>　　妙心想了一下说道：“之前岔道口有两条路，我们走了这边的一条，也许真正的主墓室在另外一边。”
<br/>　　这时候梧桐插话说道：“是有这个可能，不过，如果墓主之类的都在那边，那这一边，又是做什么的呢，总不可能费了这么大力气开凿出这一切，却什么都不放吧。”
<br/>　　大伙都觉得她说的在理，于是不得已，又回到那口水井边上，妙心只好让他们进行卢晓清那看上去不太好的计划。
<br/>　　卢晓清从墙上取下一只烛台，放在井研上，开始从包里寻找朱砂。
<br/>　　“等一下！”叶少阳眼尖，突然发现了什么，把烛台拿起来，凑到井研上的某处，仔细看去。
<br/>　　大伙不知道怎么回事，也跟着围观，这才发现井研上有一行小字。
<br/>　　字迹是刻在井研上的，很是潦草，看上去像是匆忙之间用什么硬物刻上去的，内容只有一句话：提防水中有……
<br/>　　最后两个字笔画很复杂，刻得又潦草，根本看不出是什么字。只能看到两个字各自带有“虫”字旁。
<br/>　　“这是谁刻的！”有人叫起来。
<br/>　　妙心抚摸着刻痕，喃喃道：“如果外面的石门是我祖上打开的话，这字迹很可能也是他刻下的。提防后人小心他说的那个东西。”
<br/>　　毛小方道：“你不是说，你祖先离开得很匆忙吗？”
<br/>　　“是很匆忙，但也不一定就一点时间挤不出来。”
<br/>　　“那这么说，他刻下这行字，是留给后人看的了？”叶少阳问。
<br/>　　“当然，我家世代都是地师，这古墓既然是我祖先发现的，并且葬身其中，后人当然不能不管。”妙心摆摆手说道，“我们能不能不要计较这刻痕是谁留下的了，先看看上面写的是什么。”
<br/>　　两个虫字旁的字，笔画很多，实在让人想不起来是什么东西。
<br/>　　“虫字旁……蜈蚣？”叶少阳说道。
<br/>　　大伙眼前一亮，仔细大量过去，虽然蜈蚣两个字都是虫字旁，而且蜈蚣也符合邪物的特征，不过……当大伙仔细看的时候，这才发现刻在井研上的两个字，笔画比蜈蚣要多，绝对不是蜈蚣。
<br/>　　“不是蜈蚣，那会是什么？”大伙研究了半天，又想出了好几种两个字都是虫字旁的生物，不过经过甄别，都不是。
<br/>　　“不管了！”卢晓清说道，“我们也别猜了，不管是什么邪物，总要把它弄出来再说，你们要是没意见，我就动手了。”
<br/>　　大伙只好同意。
<br/>　　于是卢晓清把一包朱砂和一包生石灰撒了下去……等两包东西落进水中之后，叶少阳想到什么，从自己的背包里拿出血渔网，罩住了井口。
<br/>　　“这是什么？”卢晓清狐疑地问。
<br/>　　“法器。”叶少阳回答了一句废话。
<br/>　　大伙趴在井研上，透过血渔网，朝下面看去，朱砂和石灰落水之后荡起的涟漪逐渐消失，但是……什么也没有发生。
<br/>　　弄错了，下面没有邪物？
<br/>　　这个念头刚在叶少阳脑海中产生，猛然间砰的一声，水花掀动，从井里突然就蹦出一个东西，一下子窜了上来，撞在了血渔网上，叶少阳本来是把血渔网捂得紧紧的，并且在上面贴了灵符加持，以为不管是什么邪物上来，至少都能抵挡一下子，结果没想到下面那个东西那么猛，一下子将叶少阳连人带血渔网都撞飞了出去，重重地摔在地上。
<br/>　　叶少阳被摔得七荤八素，刚爬起来，就看到一个庞然大物，半个身子趴在井研上，长着血盆大口，发出一声怒吼。
<br/>　　这房间本来就空旷，被这个怪物这么用力一犼，简直震耳欲聋，有一种脚下的地面都为之颤抖的感觉。
<br/>　　在场几人，全都怔住了，一时间忘了反应，那怪物一个探爪，朝离的最近的陈晓宇抓了过去。
<br/>　　陈晓宇猛然回神，哪里见过这么大块头的怪物，本能地一步跳开，把站在边上的毛小方暴露了出来。
<br/>　　毛小方也不敢硬拼，一边后退，一边摸出了一把铜离子，朝那怪物甩手打了过去，那怪物也不躲，体积又大，一把铜离子全打在了它身上，陷入皮肉中。
<br/>　　一股血浆子喷了出来。
<br/>　　不过这怪物明显没什么事，继续朝毛小方扑过去。
<br/>　　毛小方取出了青锋剑，一口气退到墙角，那怪物也飞快地爬了过来，张开大嘴，对着毛小方就咬下来，毛小方身形一矮，向边上滚了开去。
<br/>　　结果那怪物的一张鳄鱼般的长嘴没咬到他的脑袋，装在了墙壁上，岩石铸造的墙壁，居然被他撞出了一个缺口，掉下来一大片岩石碎片。
<br/>　　这力量之强，让所有人都惊呆了。
<br/>　　“快跑！先出去再说！”妙心一扬手，打出了三道灵符，飞向怪物的伸手，贴在了它的后背上。
<br/>　　“兵临地火，天下莫士，破！”
<br/>　　三道灵符燃烧起来，火光却不是往四周燃烧，而是彼此相连，像三道喷射而出的火蛇，瞬间暴涨，缠绕住了怪物庞大的身躯。
