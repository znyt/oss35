　　第676章
<br/>　　歌舒离放下崔白玉，拍马向高飞豹攻来，二人交手数招，两柄刀便狠狠的撞在一起。
<br/>　　二人都是心中一惊，歌舒离想的是，这汉子竟如此了得，自己并不能小看他。高飞豹想的是：啊呀，我恐怕不是这人对手。
<br/>　　事实上，高飞豹本身武艺并不弱，算是顶级山贼。不过，在如今蛤蟆城秦蛮、关鱼、赵子龙先后突破进入绝世境后，他的本事便算不得什么了。平日也算勤勉，经常同张肥、关鱼几人切磋，可奈何天资有限。顶级便已经是他的极限。
<br/>　　这歌舒离如果放在军中，定是战功赫赫的大将，放在崔白玉身边做亲随，当真是委屈他了。
<br/>　　对歌舒离来说，今日的经历实在憋屈，他是怎地身份，今天却被一群山贼追得如丧家犬一般。也是形势所迫，不跑也没法子，但他心里仍是不舒服。
<br/>　　恰好拿这山贼洗刀，暂消我胸中不平之气。
<br/>　　此时，崔白玉已经另外上了一匹马，一手拉着缰绳，口中喝道：“歌舒离，速战速决，不要恋战。”
<br/>　　“是。”
<br/>　　歌舒离干脆答应一声，便向高飞豹急攻而来。招招凶险，瞬间高飞豹便被对方压制住，只有招架之力，而无还手之功。
<br/>　　哎呦，不好，我可能要死！
<br/>　　高飞豹心里忽然冒出个想法，已经是心慌意乱，招式愈加狼狈。
<br/>　　“兄弟莫慌，大哥前来助你。”
<br/>　　正这时候，一匹快马奔来，斜刺里冲来，用一柄大刀，生生救下了高飞豹。
<br/>　　来人正是虎豹军的指挥官高飞虎，他们负责在城里搞事情，此刻也已经冲杀过来。
<br/>　　高飞豹在鬼门关走了一遭，此刻是呼哧呼哧喘气，口中道：“大哥，点子扎手。”
<br/>　　言外之意就是说，估计你也够呛。
<br/>　　高飞虎冷哼一声，道：“来来来，你我兄弟联手，取了这贼人的性命，也算我们立上一功。”
<br/>　　一句话激起高飞豹心中的豪横之气，他哇呀呀怪叫，抡着刀便冲了上去。
<br/>　　“贼人去死呐。”
<br/>　　三柄刀撞在一起，三匹马车轮一般旋转。高飞虎和高飞豹的武艺本身也是不弱，只是歌舒离显得更强悍些罢了。此刻以一敌二，他也是丝毫不惧，防得滴水不漏，攻得凌厉无比。
<br/>　　崔白玉皱起眉头，如此打，要打到什么时候。敌军一旦聚集，诸人逃生便变得艰难。
<br/>　　“歌舒离莫要恋战，速走。”
<br/>　　歌舒离听到了崔白玉的命令，他不是不想走，而是实在走不了。高飞豹和高飞虎死缠着不放，他怎么能轻易走脱。
<br/>　　当然，高飞虎二人想要胜过他也不容易。
<br/>　　“呵，你们兄弟真够无用，打到现在还没解决他。”
<br/>　　一名女将从一旁的胡同中窜出来。
<br/>　　高飞豹啐了口吐沫，道：“娘们不要废话，你行你试试。”
<br/>　　来人正是小白狼，她也拍马加入战团，三人齐战歌舒离。而在她加入后，歌舒离并没有感觉增加多大压力。
<br/>　　并不是说小白狼的武艺比高飞虎高飞豹兄弟二人弱，而是因为她用两柄短刀，马上武艺并不如何。其他三人都抡着大刀，她的兵器根本近不了身。
<br/>　　此刻，双方的兵丁已经打在一起，不时有人被杀，或者杀死别人。崔白玉被护在当中，神情越来越着急。如此情况，对自己越来越不利，除非，宋伯康能组织起守军，方能在绝境翻盘。
<br/>　　歌舒离也明白久战对己方不利，他想着脱身之策，注意力集中在小白狼身上。
<br/>　　她，倒是一个极好的突破口。
<br/>　　手中大刀旋转，佯装攻击高飞虎，忽然一拨马头，向小白狼撞去。小白狼吃了兵器的亏，那里招架得住，只见一柄刀向自己天灵盖砸来。
<br/>　　“去死吧。”
<br/>　　歌舒离露出狰狞可怖的笑容，此刻高飞虎和高飞豹的招式都已使老，已无援手的可能。
<br/>　　正在危急关头，突然一声大喝，有一人硬生生拦在两人中间。
<br/>　　“竖子敢尔！”
<br/>　　歌舒离的刀没有落下去，被一只大手接住，来人横眉怒目，相貌凶恶。几乎第一时间，歌舒离便知道来人是谁。
<br/>　　“程大雷！”
<br/>　　“滚！”
<br/>　　歌舒离还幻想王见王的局面，能与程大雷插招换式过上几招。如果自己能侥幸胜上一招两式，也能灭灭程大雷的威风。
<br/>　　委实说来，他想岔了。程大雷心里根本没搁他这号人物，大手一抡，直接连人带刀将他拽到马下，狠狠砸在一旁。
<br/>　　高飞豹三人都凑过来，表情有些尴尬，三人合力都没有料理掉歌舒离，最后反而要程大雷出手相救。
<br/>　　程大雷占领城主府后，既没有找到宋伯康，也没有找到崔白玉。便独自冲杀出来，正好与高飞虎三人碰上。
<br/>　　他跨在黑牛背上，目光转了一圈，最后落在崔白玉身上。
<br/>　　崔白玉目不斜视，也盯着程大雷，倒要看看这贼首，有何出奇之处。能搅得大片江山一团糊涂。
<br/>　　二人的目光在空中狠狠相撞。
<br/>　　程大雷缓缓开口：“崔白玉？”
<br/>　　崔白玉微微点头：“程当家认得我？”
<br/>　　程大雷也是刚刚知道，他点点头：“闻名久已。”
<br/>　　崔白玉骑在一匹红马上，黑袍黑纱，见面目隐藏得很好。
<br/>　　“据说，白玉美人有倾城之貌，今日本大当家倒要尝个鲜。”
<br/>　　程大雷跨着黑牛靠近，崔白玉的亲卫将她护在当中，程大雷猛吼一声：“滚开。”
<br/>　　双方兵马，不论是蛤蟆城的人，还是崔白玉的人，此刻被其气势所迫，竟不由自主的放开一条路。
<br/>　　程大雷一步步向前，与崔白玉的距离越来越近。崔白玉极力保持镇定，但她颤抖的双肩却已暴露其内心的慌张。
<br/>　　此刻，她已是程大雷的囊中之物，手到擒来。
<br/>　　“也不知道你有没有什么规矩，比如谁看见你的容貌，你就必须嫁个谁。”
<br/>　　程大雷说着，却揭崔白玉脸上黑纱。正此时，一人突然跃起，一柄匕首攻向程大雷，程大雷也是色迷心窍，竟然疏于防范，这柄刀从其胳膊划过，带出一串血花。
<br/>　　程大雷回过神，看到三名少女拦在崔白玉马前。
<br/>　　犬、鹿、刀。
<br/>(本章完)
