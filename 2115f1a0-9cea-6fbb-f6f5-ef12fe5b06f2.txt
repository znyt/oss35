        连星儿虚惊一场后，接着说道：“好像搞得差不多了，那我们就撤了吧！”
<br/>
<br/>        郎安微微点头。
<br/>
<br/>        若不是说散场，连星儿还真未发现，这一层图书室就只剩下她和郎安两个人了。
<br/>
<br/>        “千凝真是的，离开也不跟我说一声。”连星儿抱怨了一声，放了工具后，下楼时，郎安就跟在她的身后。
<br/>
<br/>        对于郎安这个贴身保镖，连星儿是一点安全感也没有，反倒对他很是提防。
<br/>
<br/>        外公名义上是说保护她，但她总觉得，外公派郎安来她身边是监视她的。
<br/>
<br/>        只因为那次在街上，她识破了郎安的身份。
<br/>
<br/>        外公是害怕她把他们的事情抖出去吗？
<br/>
<br/>        不对！
<br/>
<br/>        连星儿冥思苦想着，心里隐隐透着一丝不安。
<br/>
<br/>        倏地，她只觉脚下踏了个空，身子本能地往前倾。
<br/>
<br/>        就在她几乎快要从楼梯上栽倒下去的瞬间，一双温暖的手及时揽住了她的腰肢。
<br/>
<br/>        那双手的力度，带动着她一个旋身，让她撞了对方一个满怀。
<br/>
<br/>        在连星儿感到脑海里一片混沌的时候，郎安那没有一丝感情的声音从头顶上方传来。
<br/>
<br/>        “大小姐，小心点。”
<br/>
<br/>        连星儿在他的怀中仰头，看到的却是一双冷冽的金眸，稍纵即逝，渐变成了黑色。
<br/>
<br/>        此刻，明司寒从楼下踱步上来，刚走到拐角平台上，便看到了楼梯上，郎安搂着连星儿，两人彼此四目相对的这一幕。
<br/>
<br/>        他放在扶手上的手，在不经意间紧握成拳。
<br/>
<br/>        随后，他又慢慢地松开了手，默不吭声地转身离去。
<br/>
<br/>        连星儿是什么性子，一开始，他不早就发现了吗？
<br/>
<br/>        先是洛羽，后是网络上的他，再是现实里的他，现在又来了个郎安。
<br/>
<br/>        明司寒走后，郎安这才放开了连星儿。
<br/>
<br/>        连星儿从惊怔中缓过神来，讷讷地应了声：“谢谢。”
<br/>
<br/>        郎安没有回应，也没告诉她，刚刚他闻到了她的未婚夫的气息。
<br/>
<br/>        就在这个时候，连星儿的手机不逢其时地响了起来。
<br/>
<br/>        来电显示上是“贺泽川”。
<br/>
<br/>        连星儿犹豫了一下，才接听电话。
<br/>
<br/>        郎安倒是很自觉地背过身去。
<br/>
<br/>        “徒儿，这周星期六，我有个朋友开生日派对，我希望你能陪我去。”贺泽川直接开门见山地说道。
<br/>
<br/>        提起这周星期六，连星儿才想起，这周六也是明司寒的生日，所以，对于贺泽川的提议，她毫不犹豫地就拒绝了。
<br/>
<br/>        “我没空噢！”
<br/>
<br/>        “为什么自从我们俩见面后，你就一直在拒绝我？”贺泽川的语气听起来有些闷闷不乐。
<br/>
<br/>        而连星儿自己却没有意识到这点，反倒是被他问及的这一刻，竟然不知道该怎么解释。
<br/>
<br/>        “前些天，周末，我约你出来打游戏。你拒绝。今天，我跟你约后天的事情，你仍旧拒绝。徒儿，你是不是对为师有什么意见？如果有意见，请你直说。”贺泽川态度强硬道。
<br/>
<br/>        连星儿尴尬地笑了笑：“没有啊！我对你意见！只是真的刚好没空。”
<br/>
<br/>        “每次都这么巧吗？还是说，你只是不想见我，故意编理由。”贺泽川十分不满地质问。
<br/>
<br/>        连星儿坦然道：“星期六，我未婚夫生日。”
<br/>
<br/>        “是吗？还真巧了！”贺泽川不冷不热地说。
<br/>
<br/>        很明显，他的语气里透着对她的不信任。
<br/>
<br/>        也不知道为什么，连星儿总感觉，这见面后的师父，跟她在游戏里认识的师父是两个人。
<br/>
<br/>        想到这里。
<br/>
<br/>        连星儿却有些心不在焉地回答道：“嗯，是，确实挺巧的！我觉得，我们俩还是维持着游戏里的师徒关系吧！现实世界里，就不要以师徒相称了。也不要总是彼此联系彼此。就像以前那样，一起上线打游戏都是随缘或者在游戏里彼此给彼此留言即可。”
<br/>
<br/>        “你是觉得我在追求你？”贺泽川忽然严肃了起来。
<br/>
<br/>        连星儿汗颜道：“我没有那个意思。”
<br/>
<br/>        “既然，你已经决定如此了。那我就不打扰你了。”贺泽川说完，便把电话给直接挂断了。
<br/>
<br/>        连星儿并未计较这么多，收起手机后，下意识地看向郎安。
<br/>
<br/>        郎安的背影很修长，但浑身散发着一股不易近人的薄凉之气。
<br/>
<br/>        他背对着她，她也没喊他，而是自顾自地离开了。
<br/>
<br/>        她不过才下几步台阶，郎安便转过身来，默默地跟在了她的身后。
<br/>
<br/>        第二天是星期五，下午没课，连星儿跟顾千凝她们三说了一声，便独自一个人离开了学校。
<br/>
<br/>        明天是明司寒的生日，作为未婚妻的她，该给他挑什么礼物好？
<br/>
<br/>        连星儿在高档商场里逛了一圈，也没选中比较心仪的礼物，只得坐在商场休息区的凉椅上拿着手机在网上搜索答案。
