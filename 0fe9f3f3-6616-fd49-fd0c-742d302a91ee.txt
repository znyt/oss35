　　“杀！”
<br/>　　“杀了他！”

<br/>　　“什么？全是帝级神兵？”

<br/>　　一次性动用四件帝级神兵，就算是以阎泰五人的身份地位，亦是心中掀起惊涛骇浪。
<br/>　　因为就算是他们黄泉圣宗，帝级神兵也是屈指可说，不会超过十件，而且全部都只是帝级下品神兵而已，没有一件帝级中品神兵。

<br/>　　其他四位殿主，同样没有停顿，几乎同时出手了。
<br/>　　“魔云掌！
<br/>　　“九幽鬼王拳！”
<br/>　　“尸皇拳！”
<br/>　　“阴煞魔爪！”
<br/>　　“阴冥斩！”
<br/>　　五大强者齐齐出手，各自施展了一招强大的武技，向苏莫轰杀而去。
<br/>　　五道攻击一击比一击强大，汇聚在一起，威势简直惊天动地。
<br/>　　浩大的攻击，玄力波动浩瀚如海，几乎可以破灭一切。
<br/>　　这一次，五人俱是全力出手，四大殿主也动用了兵器，威力大增。

<br/>　　苏莫居高临下，望着急速袭来的五道攻击，面色丝毫不变。
<br/>　　“日月转！”
<br/>　　下一息，苏莫轻喝一声，立刻施展了大苍穹剑法的第三式，日月转。
<br/>　　一剑斩出，璀璨的剑气挥洒长空，从九天之上斩下。
<br/>　　一片模糊的虚引伴随着剑气而生，虚影中天地颠倒，日月倒转，玄妙异常。
<br/>　　苏莫的大苍穹剑法，只练到了勉强入门的阶段，但即便如此，作为帝级上品武学，威力也就极其强大。
<br/>　　苏莫的这一剑，斩向了阎泰的魔云掌，以及阎罗殿主的阴冥斩。
<br/>　　在苏莫出手的同时，三具魔影分身也同时出手了，俱是施展出了大苍穹剑法，分别攻向另外的三道攻击。
<br/>　　魔影分身是苏莫的玄力凝聚，受苏莫的意念控制，自然也可以施展大苍穹剑法。

<br/>　　四道剑气，威势比之五道攻击并不弱多少，锋锐之气冲天，磅礴浩大的剑威无可匹敌。
<br/>　　眨眼之间，双方的攻击正面硬撼在了一起。

<br/>　　苏莫心中一惊，身形立刻急速暴退，同时手中帝皇剑急速舞动，舞出层层剑幕，抵御毁灭般的冲击。
